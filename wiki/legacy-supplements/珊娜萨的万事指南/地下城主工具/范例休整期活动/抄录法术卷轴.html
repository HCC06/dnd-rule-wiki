抄录法术卷轴

只要有时间和耐心，一名施法者就可以将法术转变为卷轴，从而创造法术卷轴。这个活动表现了生产卷轴所需要的时间和精力。 资源 抄写魔法卷轴所需要的时间和金钱基于你希望抄录的法术等级，如法术卷轴成本表所显示。你必须同时提供法术所需要的任何材料成分。此外，你必须已经准备了这个法术，或者你想要抄录的法术在你的已知法术中。你还必须拥有奥秘技能的熟练。 如果你抄录一个戏法，卷轴将如1级施法者般工作。 表：法术卷轴成本 | | | | |-------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------| | 等级 | 成本 | 时间 | | 戏法 | 15gp | 1天 | | 一环 | 25gp | 1天 | | 二环 | 250gp | 3天 | | 三环 | 500gp | 1工作周 | | 四环 | 2,500gp | 2工作周 | | 五环 | 5,000gp | 4工作周 | | 六环 | 15,000gp | 8工作周 | | 七环 | 25,000gp | 16工作周 | | 八环 | 50,000gp | 32工作周 | | 九环 | 250,000gp | 48工作周 | 难题 制作卷轴是一项孤独的工作，一般不太可能引起人们的注意。难题的产生更有可能涉及到抄录卷轴的准备工作。 角色进行制作卷轴的每个休整期都有10%的机会引发难题，制作卷轴难题如表中示例。 表：抄录卷轴难题 | | | |---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------| | d6 | 难题 | | 1 | 你买下了所有的稀有墨水用于制作卷轴，激怒了城镇上的法师。 | | 2\* | 一个善良的神殿祭司控告你参与了黑暗魔法的交易。 | | 3 | 一名法师渴望收集在你法术书中的某项法术，并逼迫你向他售卖卷轴。 | | 4 | 由于你在创造卷轴时犯下了一个奇怪的错误，一项同等级的随机法术取代了卷轴法术。 | | 5 | 你为抄录卷轴购买的稀有羊皮纸上画着一张几乎不可见的地图。 | | 6\* | 有个小偷试图闯入你的工作室。 | \*可能涉及对手
