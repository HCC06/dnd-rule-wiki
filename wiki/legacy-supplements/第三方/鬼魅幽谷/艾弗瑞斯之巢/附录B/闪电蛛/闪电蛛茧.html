闪电蛛茧

##### 闪电蛛茧Spythronar Sac
微型异怪，无阵营
------------------------------------------------------------------------ **护甲等级** 5 **生命值** 1（1d4-1） **速度** 0尺 ------------------------------------------------------------------------ | 力量 | 敏捷 | 体质 | 智力 | 感知 | 魅力 | |-------|-------|-------|-------|-------|-------| | 1(-5) | 1(-5) | 8(-1) | 1(-5) | 3(-4) | 1(-5) | ------------------------------------------------------------------------ | | | |-------------------------------------------------------------------------------|-----------------| | **状态免疫** 目盲，魅惑，耳聋，力竭，恐慌，麻痹，石化，中毒，倒地，束缚，昏迷 | | | **感官** 震颤感知10尺（此半径外目盲），被动察觉6 | | | **语言** 无 | | | **CR** 0（10XP） | **熟练加值** +2 | ------------------------------------------------------------------------ **伪装False Appearance。**闪电蛛茧看起来就像一个由线球、树枝和泥土组成的，乱糟糟的混合体。看到蛛茧的人可以通过一个DC 15的智力（奥秘或自然）检定来识别它。 **脆弱Fragile。**进入闪电蛛茧内部的生物必须进行一次DC 10的敏捷豁免检定，否则蛛茧将被摧毁。 **释放闪电Lightning Release。**当闪电蛛茧被摧毁时，其在半径10尺内释放闪电。进入蛛茧摧毁其的生物无需进行豁免检定。该区域内的其他生物必须成功通过一次DC 10的敏捷豁免检定，否则受到4（1d8）闪电伤害。该区域内的闪电蛛群和闪电蛛网无需进行豁免，作为替代其进行下次攻击检定时具有优势。 **惊厌之诞Shocking Birth。**当闪电蛛茧受到非闪电蛛来源的闪电伤害时，其孵化，转变为拥有正常一半生命值的闪电蛛集群。集群投掷先攻并加入战斗。
