# <span style="FONT-SIZE: 26pt; FONT-FAMILY: 黑体; COLOR: maroon; mso-bidi-font-size: 48.0pt; mso-ascii-font-family: &quot;Times New Roman&quot;; mso-hansi-font-family: &quot;Times New Roman&quot;"> 第九章：战斗Combat</span>

<span lang="EN-US" style="FONT-SIZE:
 11pt; FONT-FAMILY: &quot;Cambria&quot;,serif"><span style="mso-spacerun: yes"> </span></span><span style="FONT-SIZE:
 11pt; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria">利剑击打盾牌的铿锵声。怪物爪子刺穿护甲的撕裂声。法师用法术施放烈焰凝成的火球，并爆出耀眼的闪光。刺鼻的血腥味在怪物们的恶臭中弥漫。狂怒的咆哮、胜利的欢呼、痛苦的惨叫。这就是D&D中混乱、致命、迷人的战斗场景。  
</span>

<span style="FONT-SIZE:
 11pt; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria">本章介绍了你的角色与怪物进行战斗的规则，其可以是短暂的游击，也可以是地下城中的持久战乃至更广阔的战场。本章所述的“你”指代玩家的同时也可以指代地下城主。地下城主控制所有参战的怪物和非玩家角色，而其他玩家则各控制一名冒险者。“你”也指代处于你控制中的角色或怪物。</span>
