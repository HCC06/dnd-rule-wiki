**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; COLOR: rgb(128,0,0); mso-hansi-font-family: 宋体; mso-bidi-font-weight: normal; mso-spacerun: 'yes'; mso-ansi-font-weight: bold">制作</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; COLOR: rgb(128,0,0); mso-hansi-font-family: 宋体; mso-bidi-font-weight: normal; mso-spacerun: 'yes'; mso-ansi-font-weight: bold">Crafting</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; COLOR: rgb(128,0,0); mso-hansi-font-family: 宋体; mso-bidi-font-weight: normal; mso-spacerun: 'yes'; mso-ansi-font-weight: bold"></span>**

<span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">你可以手工制作冒险装备和艺术品等非魔法物品。完成手作活动需要你拥有相关制作工具的熟练项（通常是某种工匠工具）。你也可能需要有特别的材料或在特别的地点才能进行手作活动。例如，拥有铁匠工具熟练项的某人需要熔炉才能打造刀剑或护甲。</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">进行手作活动时，你每花费一日的休整时间可以制作市面总价不超过</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">5gp</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">的一件或多件物品，而你还必须支付等于其市面总价一半的原料费用。如果你想制作物品的市场总价超过</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">5gp</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">，则你可以按每日</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">5gp</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">的进度逐日进行，直至达成其市面总价以完成制作。例如，你自己独自打造一整套全身板甲（市值</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">1500gp</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">）的话，总计需要</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">300</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">日的制作时间。</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">此外，多个角色还可以合力制作一件物品，前提是参与制作者必须都拥有所需工具的熟练项并待在同一地点进行工作。这时，每个角色每日为制作贡献</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">5gp</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">的进度。例如，三名拥有相关工具熟练项的角色在有适当设备的情况下，可以用</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">100</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">日制作一整套的全身板甲，同时还要花</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">750gp</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">的材料费用。</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">进行手作活动时，你可以维持俭朴的生活而不需支付每日</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">1gp</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">的开销，或者以半价来维持舒适的生活（详见第</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">5</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">章关于生活方式费用开支的相关内容）。</span>
