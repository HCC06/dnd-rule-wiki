## **<span style="FONT-SIZE: 22pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'">休息</span><span style="FONT-SIZE: 22pt; FONT-FAMILY: 宋体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'">Resting</span><span style="FONT-SIZE: 18pt; FONT-FAMILY: 宋体; FONT-WEIGHT: bold; mso-spacerun: 'yes'"></span>**

<span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'"> </span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">冒险者们就算有如神话英雄般神勇，也无法整日每时每刻的进行探索、社交、战斗等高强度活动。因此，他们也会需要休息——花时间吃东西和补充睡眠，还要护理身上的伤痛，并理清心绪为施法做准备，以为未来的冒险做好准备。</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'"> </span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">冒险者以及任何生物都可以在一日中任何时刻进行短休，以及以一次长休结束一日。</span>

<span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'"></span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span>

  

<span style="FONT-SIZE: 10.5pt; FONT-FAMILY: Calibri; mso-bidi-font-family: 'Times New Roman'; mso-fareast-font-family: 宋体; mso-font-kerning: 1.0000pt; mso-spacerun: 'yes'"></span>

**<span style="FONT-SIZE: 16pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'">短休</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 'Times New Roman'; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-fareast-font-family: 黑体; mso-spacerun: 'yes'">Short Rest</span>**<span style="FONT-SIZE: 10.5pt; FONT-FAMILY: Calibri; mso-bidi-font-family: 'Times New Roman'; mso-fareast-font-family: 宋体; mso-font-kerning: 1.0000pt; mso-spacerun: 'yes'"></span>

<span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'"> </span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">短休是一段短暂的休整时刻。在至少</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">1</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">小时的短休时间内，各角色只进行安静的活动，诸如饮食、阅读、护理伤口等。</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'"> </span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">完成短休时，角色可以花费一粒或多粒生命骰来恢复自身生命值，直至花光所有生命骰为止（生命骰的总数等于角色的等级）。每一粒生命骰的结果加上角色的体质调整值，即为该粒生命骰恢复的生命值（总数至少为</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">0</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">）。每掷完一粒生命骰，该角色都可以选择是否花费更多生命骰来恢复生命。一个角色已花费的生命骰，可以通过完成一次长休来恢复，其具体规则如下文所述。</span>

<span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'"></span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span>

  

<span style="FONT-SIZE: 10.5pt; FONT-FAMILY: Calibri; mso-bidi-font-family: 'Times New Roman'; mso-fareast-font-family: 宋体; mso-font-kerning: 1.0000pt; mso-spacerun: 'yes'"></span>

**<span style="FONT-SIZE: 16pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'">长休</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 'Times New Roman'; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-fareast-font-family: 黑体; mso-spacerun: 'yes'">Long Rest</span>**<span style="FONT-SIZE: 10.5pt; FONT-FAMILY: Calibri; mso-bidi-font-family: 'Times New Roman'; mso-fareast-font-family: 宋体; mso-font-kerning: 1.0000pt; mso-spacerun: 'yes'"></span>

<span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'"> </span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">长休是一段更为尽兴的休整时刻。在至少</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">8</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">小时的长休时间内，该角色可以补充至少</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">6</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">小时睡眠，以及进行不超过</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">2</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">小时的低强度活动，诸如阅读、聊天、进食、放哨等。如果休息过程中进行超过</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">1</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">小时的激烈活动（例如战斗、施法、冒险等），则该角色必须重新开始休息才能获得长休的得益。</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'"> </span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">完成长休时，该角色将恢复所有生命值，并恢复部分已消耗的生命骰（至多恢复所拥有生命骰总数的一半，其数量至少为一粒）。例如，拥有八粒生命骰的角色，他在完成长休时至多可以恢复四粒生命骰。</span><span style="FONT-SIZE: 16pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'">  
</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'"> </span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">一个角色在</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">24</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">小时内只能因长休获益一次，而且角色在开始长休时必须有最少</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: Cambria; mso-spacerun: 'yes'">1</span><span style="FONT-SIZE: 11pt; FONT-FAMILY: 宋体; mso-ascii-font-family: Cambria; mso-hansi-font-family: Cambria; mso-spacerun: 'yes'">点生命值才能从中获益。</span>
