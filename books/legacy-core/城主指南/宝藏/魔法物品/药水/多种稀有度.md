**巨人之力药水 Potion of Giant Strength**  
*药水，多种稀有度*  
饮用此药水后，1小时内，你的力量属性值将变为药水所赋予的数值（由巨人类型决定，见下表）。如果你的力量属性值已经大于或等于药水所能赋予的数值，则药水无效。  
巨人之力药水透明的药液中漂着一点对应类型巨人的指甲碎片。*霜巨人之力药水Potion of Frost Giant Strength* 和*石巨人之力药水Potion of Stone Giant Strength* 的 效果是相同的。

|                     |          |            |
|---------------------|----------|------------|
| **巨人类型**        | **力量** | **稀有度** |
| 山丘巨人Hill Giant  | 21       | 非普通     |
| 霜巨人Frost Giant   | 23       | 珍稀       |
| 石巨人Stone Giant   | 23       | 珍稀       |
| 火巨人Fire Giant    | 25       | 珍稀       |
| 云巨人Cloud Giant   | 27       | 极珍稀     |
| 风暴巨人Storm Giant | 29       | 传说       |
