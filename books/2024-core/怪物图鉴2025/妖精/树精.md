# 树精 Dryad

系于树木的自然卫士

<div class="HT">

**栖息地：** 森林；**宝藏：**任意

</div>

作为林地的守护者，树精们能魔法性地在树木、根系与枝桠之间穿梭，驱使错综的藤蔓与荆棘烦扰擅闯者。这些捉摸不透的存在大多都与它们保护的一株植物或一处天然圣域有着特殊的联系。一些树精还与它们联系最为紧密的植物共享着许多外观上的相似之处。如果它们的植物或圣域被毁，树精会变得虚弱甚至死去，只有在那些联系被治愈或魔法性地替代后才能恢复。掷一次树精圣域表或从中选择一个结果，以此启发树精的牵绊。

**树精圣域**

|         |                                                                                                    |
|---------|----------------------------------------------------------------------------------------------------|
| **1d6** | **树精栖息于此处并保护着……**                                                                       |
| 1       | 一片广袤的克隆林域，其中相同的树木彼此交织。                                                       |
| 2       | 一棵如堡垒般的巨木，如猴面包树或红衫。                                                             |
| 3       | 一株活生生的植物之锁，封印着其下的邪恶，或阻隔着往地城的通路。                                     |
| 4       | 一颗孤零零的树，屹立在狂风呼啸的山巅，或沉寂于被石化的森林之中。                                   |
| 5       | 一株有着魔法果实或奇妙种子的植物。                                                                 |
| 6       | 一只***蔓生怪Shambling Mound***或***树人Treant*** ，而树精作为妖精共生体居于其中或其侧。 |

###### 特质Traits

**魔法抗性Magic Resistance。**树精对抗法术和其他魔法效应时进行的豁免检定具有优势。  
**动植物交谈Speak with Beasts and Plants。**树精可以与野兽和植物进行交流，如同共享同一门语言。

###### 动作Actions

**多重攻击Multiattack。**树精发动一次藤鞭抽击或荆棘迸裂攻击，并使用施法施展**魅惑怪物Charm Monster** 。  
**藤鞭抽击Vine Lash。***近战攻击检定：*+6，触及10尺。*命中：*8（1d8+4）挥砍伤害  
**荆棘迸裂Thorn Burst。***远程攻击检定：*+6，射程60尺。*命中：*7（1d6+4）穿刺伤害  
**施法Spellcasting。**该树精施展以下一道法术，无需材料成分并使用魅力作为施法属性（法术豁免DC14）：  
***随意：*** **化兽为友Animal Friendship**， **魅惑怪物Charm Monster**（持续24小时；于树精再次释放该法术时提前结束）， **德鲁伊伎俩Druidcraft**  
***每项1/日：*****纠缠术Entangle，行动无踪Pass Without Trace**  

###### 附赠动作Bonus Actions

**林间飞跃Tree Stride。**如果树精位于一颗不小于大型的树的5尺内，则可以传送至该树60尺内的，另一颗不小于大型的树的5尺内的未占据空间。
