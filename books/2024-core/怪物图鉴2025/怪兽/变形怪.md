# 变形怪Doppelganger

易形渗透者

<div class="HT">

**栖息地：** 幽暗地域，城市；**宝藏：**个体

</div>

变形怪是超自然的生灵，它们能变形为一切类似人类的模样。其读心能力能帮助它们创造出近乎完美的伪装，并从毫无防护的头脑中挖掘出秘密。偶尔，变形怪也会用更公开的方式使用它们的变形能力，变为令人不安的形态来吓唬敌人。  
变形怪的行动可能和它的神秘魔法起源或更多是时候，它被雇来达成的目标有关。掷一次变形怪阴谋表或从中选择一个结果，以此启发设计一个变形怪的阴谋。

**变形怪阴谋**

|        |                                                        |
|--------|--------------------------------------------------------|
| **1d** | **变形怪在暗中策划着……**                               |
| 1      | 在那位诅咒它不得以真实形态生活的神祇的神殿中制造混乱。 |
| 2      | 隐藏一个巨大阴谋的证据。                               |
| 3      | 假扮成一个传说中的妖怪，通过恐惧来操纵一个聚居地。     |
| 4      | 取代一位贵族，享受穷奢极欲的生活。                     |

| 6      | 担任有影响力的职位，充当一场变形怪入侵计划的潜伏特工。 |

###### 动作Actions

**多重攻击Multiattack。**变形怪使用两次猛击攻击，并使用诡异面容（若条件允许）。  
**猛击Slam。***近战攻击检定：*+6（在每场战斗的第一轮中具有优势），触及5尺。*命中：*11（2d6+4）钝击伤害。  
**读心Ready Thoughts。**变形怪施展**侦测思想Detect Thoughts** ，无需法术成分并使用魅力作为施法属性（法术豁免DC12）。  
**诡异面容Unsettling Visage（充能6）。***感知豁免检定：*DC12，源自变形怪的15尺*光环*区域内的每名能看见变形怪的生物。*失败：*目标陷入*恐慌*状态，并在其每个回合结束时重复豁免，成功则终止其身上的此效应。1分钟后，其豁免自动成功。

###### 附赠动作Bonus Actions

**变形Shape-Shift。**变形怪*变形*为小型或中型的类人生物，或变回其真实形态。除体型以外，其各形态下游戏数据均相同。变形怪着装或携带的任何装备都不会随之变化。
