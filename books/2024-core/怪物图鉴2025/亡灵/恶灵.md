# 恶灵 Specter

怨怒之灵与死亡之从

<div class="HT">

**栖息地：** 幽暗地域，城市；**宝藏：**无

</div>

恶灵是被黑暗和负面情绪所吸引，吸收生命力的无形灵体。这些满怀怨恨的灵体和他们的生前没有任何联系，只是为了品尝生命和人情而去汲取凡性的内蕴。  
恶灵会去寻找有着邪恶生物存在的地点，依靠后者所激发的痛苦为食。你可以从恶灵作祟表格中选择一个或者随机一个结果来决定恶灵潜藏的地方。掷一次恶灵作祟表或从中选择一个结果，以此启发恶灵潜藏在什么样的地方。

**恶灵作祟**

|         |                                                |
|---------|------------------------------------------------|
| **1d8** | **恶灵潜藏在……**                               |
| 1       | 受怨恨、诅咒、瘟疫或惨案所折磨的社群内。       |
| 2       | 邪恶神器或致命魔法装置中。                     |
| 3       | 某个邪魔或亡灵的巢穴内。                       |
| 4       | 恶人的死亡或葬身之地附近。                     |
| 5       | 通往下层位面、负能量位面或堕影冥界的传送门旁。 |
| 6       | 死灵法师或者死亡邪教的圣所内。                 |
| 7       | 被邪恶灵魂连结的纪念碑附近。                   |
| 8       | 有过重大伤亡事件的地点。                       |

###### 特质Traits

**虚体移动Incorporeal Movement。**恶灵可以移动穿过其他生物或物件，如同穿过困难地形一般。在其回合结束时，若恶灵还处于物件内，其受到5（1d10）力场伤害  
**日照敏感Sunlight Sensitivity。**若恶灵身处阳光下，其进行的属性检定和攻击检定具有劣势。  

###### 动作Actions

**吸取生命Life Drain。***近战攻击检定：*+4，触及5尺。*命中：*7（2d6）暗蚀伤害。若目标为生物，目标的生命值上限减少等于其受到伤害的数值。
