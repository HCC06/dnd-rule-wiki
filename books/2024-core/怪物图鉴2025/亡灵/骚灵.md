# 骚灵 Poltergeist

恶意或作剧之灵

<div class="HT">

**栖息地：** 幽暗地域，城市；**宝藏：**任意

</div>

骚灵是一种喜爱叨扰或折磨生者的灵体。一般情况下它们是不可见的，但有时候也会以类似生前样貌的褪色影像出现。有些骚灵可能都没有意识到自己已然身死，只是继续重复生前的行为。还有的则是些恶意存在，或者破碎心灵的化身，为了打破和谐而在作祟之地徘徊不休。  
骚灵能自潜伏之处使用念力操控物体。掷一次骚灵活动表或从中选择一个结果，以启发骚灵以何种方式干扰生灵。

**骚灵活动**

|         |                                        |
|---------|----------------------------------------|
| **1d8** | **为了折磨生灵，骚灵决定……**           |
| 1       | 不断把别人丢掉的东西送回来。           |
| 2       | 在垂直表面留下脚印。                   |
| 3       | 发出像是有人被封在墙里的声音。         |
| 4       | 把收纳好的东西在地上重新排列。         |
| 5       | 把尸体或娃娃当成木偶一般玩弄。         |
| 6       | 在别人睡觉时把被子抽走。               |
| 7       | 在天花板上布置刀具或武器。             |
| 8       | 把书本、家具或厨具摆成匪夷所思的样子。 |

###### 特质Traits

**虚体移动Incorporeal Movement。**骚灵可以移动穿过其他生物或物件，如同穿过困难地形一般。在其回合结束时，若骚灵还处于物件内，其受到5（1d10）力场伤害  

###### 动作Actions

**多重攻击Multiattack。**骚灵发动一次掷物猛击攻击，并使用一次念力猛推。  
**掷物猛击Object Slam。***近战或远程攻击检定：*+4，触及5尺或射程30尺。*命中：*7（2d4+2）钝击伤害。  
**念力猛推Telekinetic Thrust。**力量豁免：DC12，单一30尺内骚灵可见的生物。*失败：*9（2d6+2）力场伤害，且骚灵将目标推离至多30尺。  

###### 附赠动作Bonus Actions

**消失Vanish。**骚灵获得*隐形*状态，或将之解除。
