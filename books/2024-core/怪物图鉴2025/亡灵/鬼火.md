# 鬼火 Will-o'-Wisp

幽冥之途的引路焰

<div class="HT">

**栖息地：** 森林，沼泽，城市；**宝藏：**无

</div>

鬼火在远处看来就像在黑暗中摇曳的灯盏。在废弃建筑物的窗户或者某些危险的拐角处出现，将好奇之人诱入危险之中。一旦猎物变得虚弱，它们就会吞食其生命力。  
掷一次鬼火伏击表或从中选择一个结果，以此启发鬼火如何诱害猎物。

**鬼火伏击Will-o'-Wisp Ambushes**

|         |                                  |
|---------|----------------------------------|
| **1d6** | **鬼火将猎物引诱到……**           |
| 1       | 某座即将倒塌的废弃建筑。         |
| 2       | 饥渴的食尸鬼或吸血鬼之埋伏圈。   |
| 3       | 会诅咒一切进入者的危险遗迹。     |
| 4       | 捕食生物的巢穴，比如熊或者飞龙。 |
| 5       | 某片棕霉菌\*或绿软泥\*。         |
| 6       | 某片流沙陷坑\*或薄冰\*。         |

\*详见《城主指南》

###### 特质Traits

**转瞬即逝Ephemeral。**鬼火无法着装或携带任何东西。  
**照明Illumination。**鬼火散发出半径20尺*明亮光照*以及额外20尺的*微光光照*。  
**虚体移动Incorporeal Movement。**鬼火可以移动穿过其他生物或物件，如同穿过困难地形一般。在其回合结束时，若鬼火还处于物件内，其受到5（1d10）力场伤害  

###### 动作Actions

**电击Shock。***近战攻击检定：*+4，触及5尺。*命中：*11（2d8+2）闪电伤害。  

###### 附赠动作Bonus Actions

**摄食生命Consume Life。**体质豁免：DC10，单一5尺内鬼火可见的生命值为0的生物。 *失败：*目标死亡，鬼火恢复10（3d6）生命值。  
**消失Vanish。**鬼火及其光照获得*隐形*状态，持续至其专注终止。*隐形*状态在鬼火发动攻击或使用摄食生命后立即结束。
