# 尸妖 Wight

采命之尸兵

<div class="HT">

**栖息地：** 荒漠，异界（堕影冥界），沼泽，幽暗地域，城市；**宝藏：**武具

</div>

尸妖由一些冷血兵勇的尸体转化而来，以内在的邪恶败德作为驱动力。与丧尸不同，它们保留着生前的记忆，继续实行其邪恶的计划。  
尸妖经过复生回到人间之后，仍然会继续生前的恶行，但如今则是源自对生命力的渴望。尸妖能够使用汲取活物生命的攻击。死于尸妖毒手的类人生物将在一天后复生为***丧尸Zombie***，成为尸妖顺从的手下。  
令尸妖复生的原因多种多样。掷一次尸妖动机表或从中选择一个结果，以此启发尸妖为何危害生者。

**尸妖动机**

|         |                                        |
|---------|----------------------------------------|
| **1d8** | **尸妖复生是为了...**                  |
| 1       | 在某个特定晚上挑战一切经过其坟墓的人。 |
| 2       | 征服其自认为有宣称的土地。             |
| 3       | 继续令其被判处死刑的犯罪。             |
| 4       | 追随其生前的邪恶主人。                 |
| 5       | 履行生前未竟的誓言。                   |
| 6       | 为复生它的教派或邪神效忠。             |
| 7       | 证明自己的古往今来最伟大的勇士。       |
| 8       | 寻求其被偷走的心脏或其他宝藏。         |

###### 特质Traits

**日照敏感Sunlight Sensitivity。**若尸妖身处阳光下，其进行的属性检定和攻击检定具有劣势。  

###### 动作Actions

**多重攻击Multiattack。**尸妖使用死灵剑或死灵弓发动攻击两次攻击。其可以将其中一次攻击替换为使用吸取生命。  
**死灵剑Necrotic Sword。***近战攻击检定：*+4，触及5尺。*命中：*6（1d8+2）挥砍伤害，外加4（1d8）暗蚀伤害。  
**死灵弓Necrotic Bow。***远程攻击检定：*+4，射程150/600尺。*命中：*6（1d8+2）穿刺伤害，外加4（1d8）暗蚀伤害。  
**吸取生命Life Drain。**体质豁免：DC13，单一5尺内生物。  
*失败：*6（1d8+2）暗蚀伤害，目标的生命值上限减少等于其受到伤害的数值。被此攻击杀死的类人生物会在24小时后再活化为受尸妖控制的***丧尸Zombie*** ，除非该生物被复活或其尸体被摧毁。尸妖以此法同时能控制的丧尸数上限为十二。
