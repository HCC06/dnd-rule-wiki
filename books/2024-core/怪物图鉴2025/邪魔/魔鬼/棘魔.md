# 棘魔 Spined Devil

扰乱与怀疑之魔鬼

<div class="HT">

**栖息地：** 异界（九层地狱）；**宝藏：**无

</div>

棘魔又被称为**斯皮纳魔Spinagons**；它们隐匿在下层位面的阴影之中，为它们的地狱主子搜寻秘密。棘魔偏好于从空中发起攻击，与敌人保持距离，向敌人抛掷恶毒的尖刺。棘魔收集信息，用这些秘密来控制凡人，或是引诱更强大的魔鬼。掷一次棘魔情报表，或从中选择一个结果，以此启发棘魔正在寻找或已经获取了怎样的信息。

**棘魔情报**

|         |                                      |
|---------|--------------------------------------|
| **1d6** | **棘魔正寻求关于……的信息**           |
| 1       | 神器，神器的位置以及神器的持有者     |
| 2       | 其他魔鬼或是地狱的同盟者的背叛       |
| 3       | 具有影响力的领导人所犯下的罪行或欺瞒 |
| 4       | 隐姓埋名者的真实身份                 |
| 5       | 跨位面军队的动向                     |
| 6       | 由神明隐藏起来的预言或秘密           |

###### 特质Traits

**飞掠Flyby。**棘魔飞行离开敌人的触及范围时不会引发*借机攻击*。  
**魔法抗性Magic Resistance。**棘魔对抗法术和其他魔法效应时进行的豁免检定具有优势。

###### 动作Actions

**多重攻击Multiattack。**棘魔使用地狱叉刺或尾刺发动共计两次攻击。  
**灼热叉刺Searing Folk。***近战攻击检定：*+4，触及5尺。*命中：*5（1d6+2）穿刺伤害外加3（1d6）火焰伤害。  
**地狱尾刺Tail Spine。***远程攻击检定：*+4，射程20/80尺。*命中：*4（1d4+2）穿刺伤害外加3（1d6）火焰伤害。
