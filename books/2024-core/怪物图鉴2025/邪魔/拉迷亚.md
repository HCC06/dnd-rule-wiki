# 拉迷亚 Lamia

受诅咒的交易者和废墟劫掠者

<div class="HT">

**栖息地：** 荒漠；**宝藏：**奥秘

</div>

据传说记载，首只拉迷亚是一位野心勃勃的统治者，它曾与恶魔领主格拉兹特达成了邪恶的契约，以换取永恒的权力。而作为代价，这位统治者被变为了一只拉迷亚，一种拥有狮身与诅咒之触的怪物。  
拉迷亚要么是那第一只拉迷亚的后裔，要么是曾达成了与之类似的契约。它们常栖身于废墟附近，搜寻着能令其攫取财富和影响力的神秘魔法。拉迷亚会使用魔法幻象和蛊惑手段来诱骗其他生物为它们服务。它们有时还会勾结匪徒绑架旅者，且仅当旅者接受了它们危险的契约后才会将其释放。掷一次拉迷亚契约表或从中选择一个结果，以此启发拉迷亚的渴求之物。

**拉迷亚契约**

|         |                                            |
|---------|--------------------------------------------|
| **1d6** | **拉迷亚强制缔约者……**                     |
| 1       | 为其带来某个统治者或贵族的财产。           |
| 2       | 绘制某地城或废墟的地图。                   |
| 3       | 护送其穿过临近聚落的大门。                 |
| 4       | 将一个怪异的神像放置在特定的地点或住所内。 |
| 5       | 移除某个魔法物品的诅咒，然后将其归还。     |
| 6       | 猎杀某个怪物然后取回某个特定的器官。       |

###### 动作Actions

**多重攻击Multiattack。**拉迷亚发动两次爪击攻击。其可以将其中一次攻击替换为使用腐败之触。  
**爪击Claw。***近战攻击检定：*+5，触及5尺。*命中：*7（1d8+3）挥砍伤害，外加7（2d6）心灵伤害。  
**腐败之触Corrupting Touch。***感知豁免检定：*DC13，单一5尺内拉迷亚可见的生物。*失败：*13（3d8）心灵伤害，且目标被诅咒，持续1小时。目标陷入*魅惑*和*中毒*状态，直至诅咒结束。  
**施法Spellcasting。**拉迷亚施展以下一道法术，无需任材料成分并使用魅力作为施法属性（法术豁免DC13）：  
***随意：*** **易容术Disguise Self**（可易容为大型或中型两足动物）， **次级幻象Minor Illusion**  
***每项1/日：*****指使术Geas，高级幻影Major Image，探知Scrying**  

###### 附赠动作Bonus Actions

**跳跃Leap。**拉迷亚消耗10尺移动力跳跃至多30尺。
