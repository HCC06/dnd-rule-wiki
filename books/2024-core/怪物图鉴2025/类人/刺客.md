# 刺客 Assassin

雇佣杀手 

<div class="HT">

**栖息地：** 任意；**宝藏：**器具，个体

</div>

刺客是专业的杀手，它们擅长悄悄接近受害者，在不被发现的情况下发动攻击。大多数刺客杀人都是有目的性的——有的是因为有人出钱买受害者的命，有的是为了自己的邪恶大业。刺客们擅使毒药以及其他杀人利器，并携带能助其潜入安全区域或避免抓捕的道具。 许多刺客都遵守某种职业准则，或是表现出某种标志性的怪癖。掷一次刺客作案风格表或从中选择一个结果，以此启发这名刺客有什么独特的习惯。

**刺客作案风格**

|         |                                                                      |
|---------|----------------------------------------------------------------------|
| **1d6** | **这名刺客的恶名源于……**                                             |
| 1       | 为受害者安排戏剧情节一般的“精彩”死亡。                               |
| 2       | 藏在大型物件中，例如一套铠甲，或是家具内部的空洞。                   |
| 3       | 在做案之后留下某件签名性质的物品，例如名片、鲜花、贝壳、甚至是牙齿。 |
| 4       | 冒充名人、圣人或是仆役。                                             |
| 5       | 从受害者身上收取“纪念品”。                                           |
| 6       | 使用具有某种特定颜色或气味的毒药。                                   |

###### 特质Traits

**反射闪避Evasion。**当刺客受到一个允许其进行敏捷豁免来只承受一半伤害的效应影响时，其豁免成功时不受伤害，豁免失败时只承受一半伤害。其无法在*失能*期间使用此特质。  

###### 动作Actions

**多重攻击Multiattack。**刺客使用短剑或轻弩发动共计三次攻击。  
**短剑Shortsword。***近战攻击检定：*+7，触及5尺。*命中：*7（1d6+4）穿刺伤害外加17（5d6）毒素伤害，且目标陷入*中毒*状态直至刺客的下个回合开始。  
**轻弩Light Crossbow。***远程攻击检定：*+7，射程80/320尺。*命中：*8（1d8+4）穿刺伤害外加21（6d6）毒素伤害。  

###### 附赠动作Bonus Actions

**灵巧动作Cunning Action。**刺客执行*疾走*、*撤离*或*躲藏*动作。
