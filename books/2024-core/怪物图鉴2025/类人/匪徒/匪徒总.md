# 匪徒 Bandits

恶名远扬的法外狂徒

<div class="HT">

**栖息地：** 任意；**宝藏：**个体

</div>

匪徒们蛮取豪夺。帮派成员、亡命之徒、非法佣兵等各类罪犯都可算作匪徒。然而，并不是所有匪徒都是出于贪婪而犯罪。有些匪徒是因为不公的法律、深陷绝望、甚至是受人胁迫才不得已踏入罪恶之道。  
掷一次匪徒动机表，或从中选择一个结果，以此启发并决定是什么驱使匪徒们不断犯下罪行。

**匪徒动机**

|         |                                                          |
|---------|----------------------------------------------------------|

| 2       | 是一位被抛弃的士兵，现在它要亲自夺回国家许诺给它的事物。 |
