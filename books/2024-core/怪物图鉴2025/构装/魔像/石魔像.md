# 石魔像 Stone Golem

传说与圣地的守护者

<div class="HT">

**栖息地：** 任意；**宝藏：**无

</div>

石魔像形态各异，或如古老神祇的风化雕像，或如英雄的生动塑像，亦或其创造者所构想的任何形态。无论其设计或材质如何，这些魔像皆因赋予其生命的魔法而强化，得以世代恪守创造者之命。  
石魔像通常被创造出来以守护对某个群体意义重大的场所，如重要事件的纪念碑、领袖陵墓或信仰圣所。掷一次石魔像指令表或从中选择一个结果，以此启发其遵循的指令为何。

**石魔像指令**

|         |                                                     |
|---------|-----------------------------------------------------|
| **1d6** | **石魔像遵循指令以……**                              |
| 1       | 仅允许身着仪式服饰者通过。                          |
| 2       | 对触碰城市圣物者施展**缓慢术Slow**并协助逮捕。 |
| 3       | 听命于佩戴统治者徽章者摧毁水坝或桥梁。              |
| 4       | 服从将缺失徽章置于其胸前者，随后休眠一年。          |
| 5       | 向诵念领袖遗言者揭示密道。                          |
| 6       | 警戒曾杀害其原型英雄的怪物并与之战斗。              |

###### 特质Traits

**不变形态Immutable Form。**魔像无法*变形*。  
**魔法抗性Magic Resistance。**魔像对抗法术和其他魔法效应时进行的豁免检定具有优势。  

###### 动作Actions

**多重攻击Multiattack。**魔像使用猛击或力场箭发动共计两次攻击。  
**猛击Slam。***近战攻击检定：*+10，触及5尺。*命中：*15（2d8+6）钝击伤害外加9（2d8）力场伤害。  
**力场箭Force Bolt。***远程攻击检定：*+9，射程120尺。*命中：*22（4d10）力场伤害。  

###### 附赠动作Bonus Actions

**缓慢Slow（充能5~6）。**魔像施展法术**缓慢术Slow**，无需法术成分并使用体质作为施法属性（法术豁免DC17）。
