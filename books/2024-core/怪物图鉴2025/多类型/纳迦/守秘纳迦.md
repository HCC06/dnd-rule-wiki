# 守秘纳迦 Guardian Naga

亘古长存的护识蛇灵

<div class="HT">

**栖息地：** 荒漠，森林，异界（上层位面）；**宝藏：**圣物

</div>

守秘纳迦是不朽的蛇形学者，享有着完美的记忆力。它们收集着来自与其一同生活的族群的历史与秘闻，并看护着不同文化的故事再将其准确无误地传递给新生代的后裔们。一些比共生文明更为长存的守秘纳迦仍在残垣断壁之间徘徊，铭记着已逝文明的故事，让其亡佚的子民在传承中延续下去。  
掷一次守秘纳迦秘闻表或从中选择一个结果，以此启发该纳迦的所见所闻。

**守秘纳迦秘闻**

|         |                                          |
|---------|------------------------------------------|
| **1d8** | **守秘纳迦追忆着……**                     |
| 1       | 一段古代贤者或领袖人物的遗言。           |
| 2       | 一座隐秘城市或一片大陆的位置。           |

| 5       | 一项本该被遗忘的仪式或法术。             |

| 8       | 一只传奇怪物的致命弱点。                 |

###### 特质Traits

**天界复苏Celestial Restoration。**若纳迦死亡，则除非其遗骸被施展**驱逐善恶Dispel Evil and Good** ，其在1d6天后重获生命并恢复全部生命值。  

###### 动作Actions

**多重攻击Multiattack。**该纳迦发动两次啃咬攻击。其可以将其中一次攻击替换为使用剧毒蛇涎。  
**啃咬Bite。***近战攻击检定：*+8，触及10尺。*命中：*17（2d12+4）穿刺伤害加22（4d10）毒素伤害。  
**剧毒蛇涎Poisonous Spittle。***体质豁免检定：*DC16，单一60尺内纳迦可见的生物。*失败：*31（7d8）毒素伤害，且目标陷入*目盲*状态直至纳迦的下个回合开始。*成功：*仅半伤。  
**施法Spellcasting。**该纳迦施展以下一道法术，无需姿势或材料成分并使感知作为施法属性（法术豁免DC16）：  
***随意：*****奇术Thaumaturgy**  
***每项1/日：*** **鹰眼术Clairvoyance，疗伤术Cure Wounds**（六环版本）， **焰击术Flame Strike**（六环版本）， **指使术Geas，真知术True Seeing**
