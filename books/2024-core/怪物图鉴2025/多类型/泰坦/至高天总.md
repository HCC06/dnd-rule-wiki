# 至高天 Empyreans

诸神之子嗣

<div class="HT">

**栖息地：** 任意；**宝藏：**圣物

</div>

至高天是神灵的后代。祂们并非真正的神祇，却也继承了来自其神性亲代的神性影响与威能。一些至高天类似半神，驾驭着难以想象的伟力，以及重塑凡人之能力；另一些至高天却不过是神性存在之思想或某道关注力的具现化。无论一个至高天是具体化的完美生物，还是某份神力的片影，其外表都受其创造者的影响。掷一次至高天特征表或从中选择一个结果，以此启发一位至高天继承的神力在其外表上以何种形式具现。  

**至高天特征**

|         |                                                                |
|---------|----------------------------------------------------------------|
| **1d6** | **这位至高天具有的特征是……**                                   |
| 1       | 均衡的、自然的或是仿佛注视着他人的眼瞳。                       |
| 2       | 外形色泽丰富、色调阴暗或是在不同的观者眼中有不同的相貌。       |
| 3       | 造型温柔且令人安心或是能令观者回想起美好的回忆。               |
| 4       | 其躯体是不接合的碎块、思维以可见的形似显现或是有着畸化的外形。 |
| 5       | 如机械般规律、如磐石般坚韧、外形极端对称或是如同审判者一般。   |
| 6       | 外形病态、危险、怖人或是如同观者心中之恐惧的具现。             |
