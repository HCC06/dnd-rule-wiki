**<span style="font-size:18.0pt;font-family:宋体;
color:maroon">巨龙英豪<span lang="EN-US">Heroes of the Dragon</span></span>**

  
　　并非只有从巨龙血脉中获得魔法的术士，与龙兽同行的游侠，以及习练巨龙之道的武僧才是传承了巨龙之力并获得技能、法术与职业特性的冒险者。任何职业的角色都可以与龙族有密切联系。龙类角色纽带表提供了一些点子，您可以从中找点灵感，为您的角色来点龙味儿。每个条目末尾括号中标注的职业尤其适合于该纽带，但您也完全可以自由发挥。 <span lang="EN-US">  
  
</span>**<span style="font-family:宋体;">龙类角色纽带</span>**

|                                                             |                                                                                                                                                                                              |
|-------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **<span lang="EN-US" style="font-family:宋体;">d10</span>** | **<span style="font-family:宋体;">龙类纽带</span>**                                                                                                                                          |
| **<span lang="EN-US" style="font-family:宋体;">1</span>**   | 我崇敬，并从龙神处汲取力量<span lang="EN-US"></span>最有可能的是巴哈姆特与提亚马特，但也有可能是萨迪沃或其他具备高度实力的巨龙，如阿丝特莱雅，阿莎德隆或库若尼普斯。（牧师、圣武士、魔契师） |
| **<span lang="EN-US" style="font-family:宋体;">2</span>**   | 我崇敬或被某个龙类精魂所寄宿着，就如同自然或先祖之魂一般。（野蛮人、德鲁伊、游侠）                                                                                                           |
| **<span lang="EN-US" style="font-family:宋体;">3</span>**   | 我是某个传承巨龙师祖教诲的教团之一员。（战士、武僧、圣武士、法师）                                                                                                                           |
| **<span lang="EN-US" style="font-family:宋体;">4</span>**   | 我有一位巨龙导师或宗主。（吟游诗人、德鲁伊、战士、游荡者、术士、魔契师）                                                                                                                     |
| **<span lang="EN-US" style="font-family:宋体;">5</span>**   | 我使用我的职业特性时会表现出某些龙类特征。（野蛮人、术士）                                                                                                                                   |
| **<span lang="EN-US" style="font-family:宋体;">6</span>**   | 我拥有一件龙牙、龙角、龙爪或龙鳞，并用它们作为我的法器（牧师、德鲁伊、圣武士、游侠、术士、魔契师、法师）                                                                                     |
| **<span lang="EN-US" style="font-family:宋体;">7</span>**   | 某位巨龙先祖，或给予我先祖的巨龙赋礼，让我获得了力量。（野蛮人、战士、术士）                                                                                                                 |
| **<span lang="EN-US" style="font-family:宋体;">8</span>**   | 我与自然的连结让我与某条龙建立了奇异的链接，因其存在改变了周遭自然环境的结构。（德鲁伊、游侠）                                                                                               |
| **<span lang="EN-US" style="font-family:宋体;">9</span>**   | 我最近发现我某位所爱之人或所信之人的真身是头巨龙。（任意）                                                                                                                                   |
| **<span lang="EN-US" style="font-family:宋体;">10</span>**  | 我从巨龙的宝库中偷得了某件看似平平无奇的玩意儿，但当我把它带在身上时就会发生不可思议的事情。（任意）                                                                                         |
