**<span class="15" style="FONT-SIZE:

 16pt;  COLOR: rgb(128,0,0);  mso-bidi-font-weight: normal;  mso-ansi-font-weight: bold">阶级，地位和头衔</span><span class="15" style="FONT-SIZE:

 16pt;  COLOR: rgb(128,0,0);  mso-bidi-font-weight: normal;  mso-ansi-font-weight: bold"></span>**

<span style=" FONT-FAMILY: Arial;  mso-fareast- ">正如任何严格的军事等级制度一样，军团中的每一个大地精都有一个军衔，从军阀到军官干部，再到占其大部分人数的士兵。这些军衔中最常用如下：</span><span style=" FONT-FAMILY: Arial;  mso-fareast- "></span>

<span style=" FONT-FAMILY: Arial;  mso-fareast- ">  
</span><span style=" FONT-FAMILY: Arial;  mso-fareast- ">第一等：军阀 第五等：长矛</span><span style=" FONT-FAMILY: Arial;  mso-fareast- ">  
</span><span style=" FONT-FAMILY: Arial;  mso-fareast- ">第二等：将军 第六等：拳头</span><span style=" FONT-FAMILY: Arial;  mso-fareast- ">  
</span><span style=" FONT-FAMILY: Arial;  mso-fareast- ">第三等：上尉 第七等：士兵</span><span style=" FONT-FAMILY: Arial;  mso-fareast- ">  
</span><span style=" FONT-FAMILY: Arial;  mso-fareast- ">第四等：杀人斧</span>

<span style=" FONT-FAMILY: Arial;  mso-fareast- "></span>

<span style=" FONT-FAMILY: Arial;  mso-fareast- ">一个军团由称为旗的单位组成，每一旗都由一组相互关联的家庭组成。</span><span style=" FONT-FAMILY: Arial;  mso-fareast- "></span>

<span style=" FONT-FAMILY: Arial;  mso-fareast- ">一个旗的成员一起生活、工作和战斗，每一个旗在军团中都有各自的地位，这反映在其军官的权力上。例如，级别最高的旗的队长可以要求级别较低的任何一个旗的队长遵循他们的命令。</span><span style=" FONT-FAMILY: Arial;  mso-fareast- "></span>

<span style=" FONT-FAMILY: Arial;  mso-fareast- ">级别和责任在不同军团甚至同一军团不同的旗之间并不一定是相同的。在一个军团中，由一个上尉率领的步兵方阵可能有二百人，而在另一个军团中，这样的部队只有二十人。一个旗的拳头可能统领只有四个座狼骑士，而同一军团的另一个旗里一个拳头可能率领十个。如果一个等级在军团中不起作用，军阀会将其从等级中删除以保证效率最大化。</span><span style=" FONT-FAMILY: Arial;  mso-fareast- "></span>
