------------------------------------------------------------------------

<span style=" FONT-FAMILY: Arial;   mso-fareast- ">夺心魔很少使用非类人生物作为奴仆或者和他们发展其他的关系。他们中的大多数要么太大、太强壮以至无法长期关入栏中，要么智力太有限以至只能完成最简单的任务。一般来说，在夺心魔的左右发现的非类人生物都是灵吸怪为了特定的目的创造或育成的。这些生物中有几种值得特别提及。  
  
</span>**<span style="FONT-SIZE:

 12pt; FONT-FAMILY: Arial; FONT-WEIGHT: bold; COLOR: rgb(128,0,0);  mso-fareast- ">噬脑怪</span><span style="
  FONT-WEIGHT: bold; COLOR: rgb(128,0,0);  "> Intellect Devourer</span><span style="
  FONT-WEIGHT: bold; COLOR: rgb(128,0,0);  "></span>**

<span style=" FONT-FAMILY: Arial;   mso-fareast- ">几乎每一个夺心魔社群都会制造出噬脑怪，并在巢穴周围安置一些，以保持警惕，杀死入侵者，并引诱新的受害者走向灭亡。</span>

<span style=" FONT-FAMILY: Arial;   mso-fareast- "></span><span style=" FONT-FAMILY: Arial;   mso-fareast- "></span>

**<span style="FONT-SIZE:

  FONT-WEIGHT: bold; COLOR: rgb(128,0,0);  "> Mindwitnesses</span><span style="
  FONT-WEIGHT: bold; COLOR: rgb(128,0,0);  "></span>**

<span style=" FONT-FAMILY: Arial;   mso-fareast- ">证心者代表了夺心魔典型的繁殖模式的一个例外。如果一个社群成功地捕获了一只眼魔，它可以用蝌蚪把这只生物变成一种奇特的杂交生物，成为“</span><span style=" FONT-FAMILY: Arial;   mso-fareast- ">证心者”</span><span style=" FONT-FAMILY: Arial;   mso-fareast- ">。证心者是一种灵能中枢，能够收集并强化灵吸怪的灵能力量。</span>

<span style=" FONT-FAMILY: Arial;   mso-fareast- "></span><span style=" FONT-FAMILY: Arial;   mso-fareast- ">有关证心者的更多信息，请参阅本书第3</span><span style=" FONT-FAMILY: Arial;   mso-fareast- ">章。</span><span style=" FONT-FAMILY: Arial;   mso-fareast- "></span>

<span style=" FONT-FAMILY: Arial;   mso-fareast- ">这些恐怖的生物，被夺心魔所憎恨，有时会在那些缺乏夺心魔学识的人摧毁一个社群时产生。当蝌蚪池无人照料时，就会产生夺心巨虫。蝌蚪们互相攻击，存活下的蝌蚪会长得很大。与紫虫相似，这些庞然大物</span><span style="FONT-SIZE:
