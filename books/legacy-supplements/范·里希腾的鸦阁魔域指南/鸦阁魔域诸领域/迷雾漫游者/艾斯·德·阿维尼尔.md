### **<span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;">艾丝德阿维尼尔</span><span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;"> Ez d</span><span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;">'</span><span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;">Avenir</span><span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;"></span>**

<span style="">艾丝原名艾丝美拉达拉达纳维奇</span><span style="mso-bidi-font-family: 'Times New Roman'">Ezmerelda Radanavich</span><span style="">，但她更喜欢用“艾丝”这个名字。这名年轻的漫游者最初见到的怪物是在自己家人中间，她的家族精于操纵，假扮成维斯塔尼人谋害旅行者。最终，他们绑架了伊拉斯谟范里希滕，把他卖给吸血鬼梅图斯男爵，伊拉斯谟死于男爵之手。在和伊拉斯谟相处的一小点时间里，艾丝和他成为了朋友。她听伊拉斯谟充满深情地讲述自己的家庭，这样的家庭和艾丝所知的截然不同。当伊拉斯谟的父亲鲁道夫范里希滕追上她的家族、制裁她母亲之时，艾丝没有阻止。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">接下来的几年里，艾丝加入了一支维斯塔尼人的队伍，得到了艾丝德阿维尼尔这个名字。她走了很远，却从没找到像伊拉斯谟描述中那样的归属之地。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">最终，她找到了那个能告诉她更多的人：鲁道夫范里希滕。经过谨慎而周到的自我介绍，艾丝见到了范里希滕，并跟随他修行了一段时间，拼尽全力地学习如何猎杀致命的生物。虽然艾丝和鲁道夫性格不合，但她却意外地重新和伊拉斯谟取得了联系。伊拉斯谟现在是一个的</span>**<span style="  FONT-WEIGHT: bold; ; ;  ">幽魂</span><span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'">ghost</span>**<span style="">，两人重燃友谊，最终携手找到了超越鲁道夫范里希滕执念的道路。艾丝不愿让自己和鲁道夫医生的关系恶化，于是她离开了莫登特郡，去追猎邪恶，也去追寻属于自己的家庭。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">从那时起，艾丝就变了许多。她探索迷雾</span><span style="mso-bidi-font-family: 'Times New Roman'">Mists</span><span style="">的奥秘，在被狼人袭击之后将一条腿换成了工艺杰出的假肢。艾丝希望她的探索之旅和她老导师智慧最终能让她创造出一处足够安全的地方一个被称为“家”的地方。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>
