#### **<span style="FONT-SIZE: 12pt;  FONT-WEIGHT: bold; ;">灵魂回荡</span><span style="FONT-SIZE: 12pt;  FONT-WEIGHT: bold; ;">Echoing Soul</span><span style="FONT-SIZE: 12pt;  FONT-WEIGHT: bold; ;"></span>**

<span style="">你的灵魂并不只属于你，或者说，并不总是只属于你。可能是你拥有前世，也可能是你的灵魂被交换到了另外一个身体里去，又或者是你和另外一个存在有所连接，你能感受到来自另一个生命的回响。这样的回响赋予你知识，才能，甚至语言。你无法解释这种现象，但有时他们会出乎意料地干涉你的看法和你的信念。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">从灵魂回响表骰个d6或者在其中选择一个选项作为你灵魂回荡的特质。除此之外，你还会获得以下列出的特质</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">d6 灵魂回响表</span>**<span style="mso-bidi-font-family: 'Times New Roman'">Soul Echoes</span>**<span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'"></span>**

<span style="">1 我的灵魂与其他地方的人相连那个人或许是在另外一个领域，世界乃至于位面。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">2 我已经活了很多世了。每当我死亡的时候灵魂都会转世而不是继续前往下一个阶段。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">3 我原本的意识已经从我原本的身体中移除并植入到了这个新的身体</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">4 我在身体和精神上已经和另外个人合为一体了</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">5 我与一种无形而又超凡脱俗的力量共享一具身体</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">6 时空在我身边被撕裂，我有的时候会与自己的过去和未来发生些冲突</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">通灵能力</span>**<span style="mso-bidi-font-family: 'Times New Roman'">Channeled Prowess.</span><span style="">你可以额外选择两项熟练技能</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">原初之声</span>**<span style="mso-bidi-font-family: 'Times New Roman'">Inherent Tongue.</span><span style="">你可以说，读，写你选择的另外一门语言</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">恼人回音</span>**<span style="mso-bidi-font-family: 'Times New Roman'">Intrusive Echoes.</span><span style="">当你在进行攻击检定，属性检定或豁免检定的d20上骰出1时，你的灵魂记忆会即刻浮现出来并且降临在你的身上盖过你的感知和体验。你可能会把你周围的人看成一个不同于他们的人，或者会因为对周边的世界的拥有双重体验而昏头转向。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">掷一个恼人回音表，来决定回声带来的这些生动的记忆以及感知的效果。一旦恼人回音发生，直到你完成一个短休或一个长休前都不会再次出现。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">d6 恼人回音表</span>**<span style="mso-bidi-font-family: 'Times New Roman'">Intrusive Echoes</span>**<span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'"></span>**

<span style="">1 你被一个你能看到的生物（由D</span><span style="mso-bidi-font-family: 'Times New Roman'">M</span><span style="">决定）</span>**<span style="  FONT-WEIGHT: bold; ; ;  ">魅惑</span>**<span style="">1分钟直到该生物伤害你</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">2 你被一个你能看到的生物（由D</span><span style="mso-bidi-font-family: 'Times New Roman'">M</span><span style="">决定）</span>**<span style="  FONT-WEIGHT: bold; ; ;  ">恐慌</span>**<span style="">1分钟直到该生物伤害你</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">3 在你的下一个回合结束之前，你感官会感知到你周围另外一个时空或地点，在此期间你对你周围的实际环境处于</span>**<span style="  FONT-WEIGHT: bold; ; ;  ">目盲</span>**<span style="">状态</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">4 你感知到另外一个不同的时空或地点与你周围的环境重叠在一起。你的速度减半直到你的下个回合结束。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">5 各种记忆与感觉淹没了你。直到你的下个回合开始前你处于</span>**<span style="  FONT-WEIGHT: bold; ; ;  ">失能</span>**<span style="">状态</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">6 你想起的记忆是一次光荣的胜利。你可以重骰这次攻击检定，属性检定或豁免检定。你必须使用新的掷骰结果。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>
