#### **<span style="FONT-SIZE: 12pt;  FONT-WEIGHT: bold; ;">死亡之触</span><span style="FONT-SIZE: 12pt;  FONT-WEIGHT: bold; ;">Touch Of Death</span>*<span style="FONT-SIZE: 12pt;  FONT-WEIGHT: bold; FONT-STYLE: italic; ;"></span>***

<span style="">你的触摸会给他人带来痛苦，会伤害你接触到的任何人，你内在的死亡力量是你无法所掌握的。折磨着任何接触你裸露的皮肤的人。同样，你可以用你的触摸把死亡送到你的敌人身上。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">从致命触摸表中骰取或选取一个选项，来确定如何呈现这个黑暗赠礼以何种形式表现出来。此外，你还会获得相应特征。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">致命触摸表</span><span style="mso-bidi-font-family: 'Times New Roman'">Deadly Touch</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">d6 表现</span><span style="mso-bidi-font-family: 'Times New Roman'">Manifestation</span>**<span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'"></span>**

<span style="">1 你的身体被炼金术或生物实验所改变，实验给你注入了致命的化学物质。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">2 被你轻视的德鲁伊或妖精施放了魔法，导致在你出现时，周围的小型植物枯萎，昆虫死亡。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">3 你会带来一个可怕的预兆。任何被你的触摸所伤害的生物都会出现暂时性的伤痕，以此被一个团体，恶魔，神灵或其他对你感兴趣的力量所标记。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">4 你本该死在一次致命的经历里，但你幸存了下来。从那次之后，死亡和邪恶就常伴你身边。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">5 你与时间或现实不同步。你所接触的任何东西都会被扭曲。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="">6 每当你触摸一个生物，你都会听到你曾经所伤害过的人的笑声</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">死亡之触</span><span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'">Death Touch. </span>**<span style="">你可以集中精神，用你的致命的触摸对抗你的敌人。以一个动作，你进行一次徒手攻击。命中时，目标会额外受到1d10暗蚀伤害。当你达到5级时，此伤害增加1d10，变为（2d10）,11级时变为（3d10）,17级时变为（4d10）。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">无法逃避的死亡</span><span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'">Inescapable Death</span>**<span style="mso-bidi-font-family: 'Times New Roman'">.</span><span style="">当你以攻击命中目标并造成暗蚀伤害时，你忽略目标对该伤害的抗性。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">枯萎连接</span><span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'">Withering Contact.</span>**<span style=" FONT-FAMILY: Roboto; BACKGROUND: rgb(249,249,249); ;   mso-shading: rgb(249,249,249)"> </span><span style="">你的回合开始时，若你擒抱着一个生物，或你被一个生物所擒抱，该生物会受到1d10的暗蚀伤害。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>
