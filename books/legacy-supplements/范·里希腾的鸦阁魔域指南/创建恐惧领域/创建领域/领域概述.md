### **<span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;">领域概述</span><span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;">Domain Overview</span><span style="FONT-SIZE: 13.5pt;  FONT-WEIGHT: bold; ;"></span>**

<span style="">该领域的建筑、自然元素和地标都代表了黑暗领主卑鄙的过去。在发展这些特征时，请遵循以下步骤。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

<span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">指定地点</span><span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'">Specify Locations.</span>**<span style="">选择与你的黑暗领主的过去类似的地点，或者从本章后面的</span><span style="mso-bidi-font-family: 'Times New Roman'"> "恐怖</span><span style="">题材</span><span style="mso-bidi-font-family: 'Times New Roman'"> "一节中的设置表中获得灵感。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">结合地点和内涵细节</span><span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'">Combine Locations and Visceral Details.</span>**<span style="">用几句话来描述这个领域最突出的地点是如何被扭曲的，这些建议是在回答</span><span style="mso-bidi-font-family: 'Times New Roman'"> "黑暗领主的阴影 "部分的问题时产生的。把对立面放在一起，把恐怖的东西和可爱的东西放在一起，把死人和活人放在一起，把威胁和欢迎放在一起。以出乎意料的方式将细节混合在一起，越是不自然或令人反感的越好。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>

**<span style="  FONT-WEIGHT: bold; ; ;  ">合理化和正常化</span><span style="  FONT-WEIGHT: bold; ; ;  mso-bidi-font-family: 'Times New Roman'">Rationalize and Normalize.</span>**<span style="">概述居民用来假装没有问题的理由。什么可以合理地解释他们的超现实经历？他们声称血河是由于天然的粘土沉积物造成的深红色，沼泽中的恶魔之光不过是沼泽气体，而目前的隐居者是该地区最初的隐居者的后代。该地区的许多居民可能是黑暗势力的创造物，该地区是他们所知道的唯一现实，但一个局外人可能需要做哪些逻辑和心理上的跳跃才能在晚上睡着。</span><span style="mso-bidi-font-family: 'Times New Roman'"></span>
