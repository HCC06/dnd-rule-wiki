# 邪魔雕像 Fiendish Icon

地狱魔法驱动的雕像

<div class="HT">

**栖息地：** 城市；**宝藏：**无

</div>

邪魔雕像是一种怪诞、具有智慧的雕像，刻画的是可怖的邪魔形象。这些生物因邪恶仪式而获得生命。邪魔雕像能喷出地狱之火来炙烤敌人，而且它们常常伪装成普通的雕像，直到跃起伏击入侵者。  
掷一次邪魔雕像外观表或从中选择一个结果，以此启发某个特定邪魔雕像的形象。

**邪魔雕像外观**

|---------|------------------------------------------------------------|
| **1d6** | **邪魔雕像被刻画为……**                                     |

| 3       | 一个身材敦实、身披长袍的身影，一条分叉的尾巴从衣袍下探出。 |
| 4       | 一个长着锋利獠牙的小天使，伸出一只手作欢迎状。             |

| 6       | 一个长角邪魔，模样与某位角色极为相似。                     |

**魔法抗性Magic Resistance。** 雕像对抗法术和其他魔法效应时进行的豁免检定具有优势。

**多重攻击Multiattack。**雕像发动两次猛击攻击并使用炽热爆发。  
**猛击 Slam。***近战攻击检定：*+4，触及5尺。*命中：*4（1d4+2）钝击伤害。
