## 丰收领域 Harvest Domain

在丰收循环中播种、生长与收获

丰收领域的神圣力量源自一种循环：始于播种创造，盛于蓬勃丰盈，终于收割肃清，而后再度轮回不息。司掌农耕与作物的神祇可能会专注于丰收循环中的某一阶段，但大多数都会从整个循环中汲取力量。该领域的牧师被自然的循环本质所吸引，经常扮演不同循环的牧者角色。

*3级：地方历法 Community Almanac*  
你获得一套草药工具，并拥有草药工具的熟练。你也会获得自然或求生技能的熟练。

*3级：丰收领域法术 Harvest Domain Spells*  
当你完成一次短休或长休时，选择丰收循环的一个阶段：**播种Sowing**、**生长Growing**或**收割Reaping**。见丰收领域法术表；当你达到特定的牧师等级时，你就始终准备着表中所选阶段对应的法术。

*3级：丰饶之角 Cornucopia*  
当你开始一次短休时，如果你没有陷入失能状态，你可以魔法般地创造一个丰饶之角，里面装满足以供给数量等同于你熟练加值两倍的生物食用的食物。当食用这些食物的生物完成短休时，你可以将数量等于你的牧师等级的d4骰分配给这些生物。所有生物立刻掷出分配给它们的d4骰并恢复等于结果总值的生命值。当短休结束时，丰饶之角以及任何未被食用的食物都会消失。  
一旦你使用了此特性，则直至你完成一次长休前，都无法再次使用。

*3级：五谷丰登 Bountiful Harvest*  
你可以使用魔法动作展示你的圣徽并消耗一次引导神力Channel Divinity使用次数，唤起丰收之灵。你周边60尺范围内至多6个由你选择的生物获得英雄激励。一个生物必须完成一次长休才能再次通过该特性获得英雄激励。  
此外，你赋予所有目标如下增益之一。如果你当前的丰收阶段与增益所对应的不同，则其变为增益所对应的阶段。

- <div>

  **播种Sowing。** 目标获得等同于你熟练加值加上你感知调整值的临时生命值。

  </div>

- <div>

  **生长Growing。** 直至你的下一回合结束，当目标进行攻击检定或属性检定时，其会在结果总值中加入1d4。

  </div>

- <div>

  **收割Reaping。** 目标在你的下一回合结束前首次伤害一个生物时，会额外造成等同于你熟练加值的暗蚀伤害。

  </div>

*6级：四时祝福 Blessing of the Seasons*  
你可以引导丰收循环之力，来化解潜在的灾难。当你周边30尺范围内一个你可见的生物失败于一次豁免检定（该豁免的类型取决于你当前的丰收阶段）时，你可以使用反应允许其重掷该d20并使用新的结果。你可以影响的豁免检定类型如下：

- <div>

  **播种Sowing。**敏捷和魅力。

  </div>

- <div>

  **生长Growing。**体质和智力。

  </div>

- <div>

  **收割Reaping。**力量和感知。

  </div>

此外，无论该豁免检定成功还是失败，掷数量等于你熟练加值的d4骰。目标恢复等于结果总值的生命值。  
你能使用本特性的次数等于你的感知调整值（至少为1）。完成一次长休后，你重获所有已消耗的使用次数。

*17级：永恒之轮 Eternal Wheel*  
当你使用你的五谷丰登时，你可以唤起生死轮回的无尽循环，为你自己以及受你的引导神力影响的生物提供基于你当前丰收阶段的一项额外增益：

- <div>

  **播种Sowing。**目标获得**再生术Regenerate**的效应，持续1分钟或直至你失去专注。

  </div>

- <div>

  **生长Growing。**目标在豁免检定和属性检定中具有优势，持续1小时或直至你失去专注（如同专注一个法术）。

  </div>

- <div>

  **收割Reaping。**目标获得暗蚀、心灵和毒素伤害抗性，持续1小时。

  </div>

一旦你使用了此特性，则直至你完成一次长休前，都无法再次使用。你也可以消耗一枚七环或更高环阶的法术位（无需动作）来恢复该特性的使用次数。

### 丰收仪式 Rites of the Harvest

使用丰收仪式表来获得关于各个丰收循环阶段所对应的庆典的灵感。  
*丰收仪式Harvest Rites*

**播种仪式Sowing Rites**

|                                                                    |
|--------------------------------------------------------------------|
| 把祭品埋在新犁过的田里，比如干玉米壳、死鱼、知更鸟蛋或野兔脚       |
| 酿造受祝福的药水，用于饮用或灌溉，以提高作物、牲畜或人类的生育能力 |
| 在旅程开始时赠予旅行者一小袋种子，以确保他们顺利抵达目的地         |

**生长仪式Growing Rites**

|                                                                        |
|------------------------------------------------------------------------|
| 以自然疗法照料病人或伤者，使用纯净水源、沃土、肥灰或果泥敷料以加速恢复 |
| 为农作物或牲畜祈福，祈求风调雨顺，使作物茁壮、牲畜繁盛                 |
| 主持婚礼，以编织的谷杆或亚麻绳将新婚夫妇系结在一起                     |

**收割仪式Reaping Rites**

|                                                                              |
|------------------------------------------------------------------------------|
| 举行葬礼，将逝者的遗体或骨灰归葬田野，混入其家乡的泥土和腐烂的作物以肥沃土壤 |
| 在战斗前或收获前，打磨经由丰收之神祝福的刀刃，以确保所有收获皆归献于神圣     |
| 屠宰成年牲畜或收割成熟作物，以庆祝大自然的恩赐                               |
