海洋环境<span lang="EN-US">OCEAN ENVIRONS</span>

对于那些陆地上的居民来说，海洋是一个充满难以预知的危险的陌生世界。然而，埋藏在海底的奇珍异宝使得英勇和毫无顾忌的水手们甘愿冒这些风险。本节详细介绍了人们在海面上和波涛之下可能遭遇的各种环境特征。

<span lang="EN-US"></span>

蓝洞<span lang="EN-US">BLUE HOLES</span>

蓝洞是在浅水的海床上形成的圆形沉积空洞。洞中深蓝色的海水与周围浅蓝色的海水形成了鲜明的对比。蓝洞的直径为<span lang="EN-US">1d10 x 100</span>英尺，深度为<span lang="EN-US">1d10 x 100</span>英尺。

珊瑚礁<span lang="EN-US">CURRENTS</span>

珊瑚礁遍布海底，虽然大部分都是在距离海岸线<span lang="EN-US">150</span>尺深度以内发现的。珊瑚礁的长度和宽度从几尺到<span lang="EN-US">1000</span>多英里不等。这些丰富多彩的生态系统容纳了小鱼、甲壳类动物以及软体动物，导致掠食者们到珊瑚礁中寻找猎物。

参差不齐的尖锐珊瑚礁会伤害那些被迫进入它们的家伙。当一种生物并非使用自身的动作、附赠动作或者反应被迫在珊瑚礁中移动时，每在珊瑚礁中被移动过<span lang="EN-US">5</span>尺就受到<span lang="EN-US">3</span>（<span lang="EN-US">1d6</span>）点伤害。

<span lang="EN-US"></span>

洋流<span lang="EN-US">CURRENTS</span>

海水在风和潮汐的作用下流动，形成一种称为洋流的恒定河流状水流。潮汐流通常是最强的，它们通常在距离海岸<span lang="EN-US">50</span>英里以内和少于<span lang="EN-US">300</span>尺内。这些洋流的速度为每小时<span lang="EN-US">1d6</span>英里，每日都在变化着。

洋流在出海越远，在海洋越深的地方就越弱。这些洋流的速度为每小时<span lang="EN-US">1d4-1</span>英里，每日都在变化。

<span lang="EN-US"></span>

洋流与旅行<span lang="EN-US">CURRENTS AND TRAVEL</span>

当一艘船只或生物以与洋流相同的方向行进时，这条洋流的速度会加到船只或生物的行进速度中。逆流而行的生物或船只，其速度等于从该生物或船只的行进速度中减去的洋流之后的速度。如果这导致速度为负值，则船只或生物可以让水流携带它们前进。或者，船上的船员或生物可以通过划船或用力游泳来逆流航行，以船或生物正常速度的一半移动。与水流搏斗的船只和生物必须在每小时结束时进行一次体质豁免检定，加值等于船员团队的素质值（如果他们有船员的话）。失败的船员的素质值下降<span lang="EN-US">1</span>。失败的生物会获得一级力竭。这一节豁免检定<span lang="EN-US">DC</span>是<span lang="EN-US">10+</span>与洋流搏斗连续花费的小时数。

<span lang="EN-US"></span>

被洋流淹没<span lang="EN-US">SUBMERGED I N CURRENTS</span>

当一个没有游泳速度的生物被洋流淹没时，该生物必须在回合结束时进行一次力量（运动）检定，或者沿洋流方向被冲走<span lang="EN-US">1d4 x 5</span>英尺。这次检定的<span lang="EN-US">DC</span>等于<span lang="EN-US">10+</span>洋流每小时移动英里数。

<span lang="EN-US"></span>

深海<span lang="EN-US">DEPTH</span>

居住在海面以上的生物越是深入海底，环境就变得越发恶劣。潜入深海者必须谨记这些自然的力量。

<span lang="EN-US"></span>

光照<span lang="EN-US">ILLUMINATION</span>

根据光照，海洋可以被分为三层。

**阳光带<span lang="EN-US">Sunlight Zone.</span>**

阳光带从海面延伸到<span lang="EN-US">650</span>英尺的深度。这片海域的自然光照条件与海面一致。

**晨昏带<span lang="EN-US"> Twilight Zone.</span>**

晨昏带在<span lang="EN-US">650</span>尺到<span lang="EN-US">1000</span>尺深度之间。当上面的阳光带沐浴在自然的明亮光照中时，微光区充满了昏暗的光线。如果阳光带充满了昏暗的光线或是黑暗时，微光带就是一片漆黑的。

**午夜带<span lang="EN-US">Midnight Zone.</span>**

任何<span lang="EN-US">1000</span>英尺往下的深度都视为午夜带，因为没有自然光从海面透入到这么深的地方。

<span lang="EN-US"></span>

压力与温度<span lang="EN-US">PRESSURE AND TEMPERATURE</span>

水压，或者生物或物体头顶水的重量，随着深度的增加而增加。相反，温度随着深度的增加而降低。压力和温度对<span lang="EN-US">100</span>英尺往下没有游泳速度的生物的影响可以见《城主指南》第五章中“非常规环境<span lang="EN-US">Unusual Environments</span>”。

幽灵雾<span lang="EN-US">GHOST FOG</span>

幽灵雾中包裹着被扔进海中的受害者的灵魂。通过一个<span lang="EN-US">DC15</span>的智力（奥秘）检定能够发现雾气是幽灵雾。

死者的灵魂希望生者加入它们。一艘船每沉浸在幽灵雾中一个小时，进行一次掷骰。是偶数的场合，<span lang="EN-US">2d4</span>个敌对的**幽灵<span lang="EN-US">Specter</span>**出现并袭击船上的人。如果一个类人生物在幽灵雾中死去，它的灵魂会化作一个对非亡灵的生者都充满敌意的**幽灵<span lang="EN-US">Specter</span>**。

所有处于幽灵雾中的非亡灵对暗蚀伤害具有易伤。

<span lang="EN-US"></span>

堕影冥雾<span lang="EN-US">SHADOWFELL FOG</span>

堕影冥雾在两个位面之间边界的薄弱点渗入物质位面。通过一个<span lang="EN-US">DC15</span>的智力（奥秘）鉴定能发现雾气是堕影冥雾。

当船只穿过这令人压抑、阴影笼罩的迷雾时，它的船员和乘客都会感到绝望。船只没浸没在雾气中一个小时，船员都必须掷一个<span lang="EN-US">D20</span>并将其素质值加到结果中进行素质值检定。该检定的<span lang="EN-US">DC</span>为<span lang="EN-US">10+</span>船只浸在堕影冥雾中的小时数。如果船员没有通过这一检定，他们的素质值在通过堕影冥雾后的一天之内下降<span lang="EN-US">1</span>。

<span lang="EN-US"></span>

狂野魔法迷雾<span lang="EN-US">WILD MAGIC FOG</span>

很少有人知道狂野魔法迷雾的起源，一些人声称它是反复无常的神灵们的造物，而另一些人则认为它是魔法实验出错的结果。通过一个<span lang="EN-US">DC15</span>的智力（奥秘）检定能发现雾气是狂野魔法迷雾。

每当一个处于狂野魔法迷雾中的生物施放一个一环或者更高环的法术时，从《玩家手册》第三章“职业”中的狂野魔法浪涌<span lang="EN-US">Wild Magic Surge</span>表格中掷骰，来创造出一种魔法效应。

海藻森林<span lang="EN-US">KELP FORESTS</span>

海藻生长在海底茂密的森林之中。大多数森林位于距离海岸线不到<span lang="EN-US">100</span>英尺的深度，长度和宽度从<span lang="EN-US">10</span>英尺到<span lang="EN-US">100</span>英里。海藻可以长成长达<span lang="EN-US">175</span>英尺的长蔓。

海藻的根很浅，一整片森林都能被一场暴风雨连根拔起，但它的生长速度很快，有时一天高达<span lang="EN-US">18</span>英寸，这意味着海带森林可以迅速生长。这样的森林可以藏匿数千条小鱼和其他捕食动物，吸引捕食者。森林是如此的茂密，以至于对<span lang="EN-US">10</span>英尺以外的观测者对它当中的一切都处于重度遮蔽，整个地区都是困难的地形。

<span lang="EN-US"></span>

克拉肯之墓<span lang="EN-US">KRAKEN's GRAVE</span>

当克拉肯死后，它们的躯体常常在海底腐朽。在这种情况下，地面吸收了腐烂的克拉肯的超自然能力，用克拉肯身体形状的黑色污痕标记这一区域。这种污痕叫做“克拉肯之墓”。

当一个生物移动到克拉肯之墓<span lang="EN-US">30</span>尺以内，或者在这一范围内开始其回合时。该生物必须成功通过<span lang="EN-US">DC14</span>的敏捷（隐匿）检定，否则会惊扰坟墓。一旦一个生物惊扰到了坟墓，一条由闪电构成的触须会从地面伸出，并以<span lang="EN-US">+7</span>加值攻击这个生物。命中则目标受到<span lang="EN-US">10</span>（<span lang="EN-US">3d6</span>）闪电伤害，且它必须成功通过<span lang="EN-US">DC14</span>体质豁免，否则会在接下来<span lang="EN-US">2d4</span>小时内陷入目盲。

<span lang="EN-US"></span>

诱饵光<span lang="EN-US">LURE LIGHTS</span>

当底栖魔鱼死亡时，它们的灵魂有时会聚集成一团，被称为诱饵光。这些直径<span lang="EN-US">100</span>英尺的微弱黄色光芒聚集在<span lang="EN-US">1000</span>英尺的深海中。它们以<span lang="EN-US">100</span>尺为半径发出明亮光照，再外围<span lang="EN-US">100</span>尺范围内发出微光。

任何看到诱饵光的生物必须成功通过<span lang="EN-US">DC14</span>感知豁免否则被光芒魅惑<span lang="EN-US">24</span>小时或指导光芒被破坏。一个成功通过豁免的生物在<span lang="EN-US">24</span>小时内免疫诱饵光的效应。

当被光芒魅惑时，生物无法主动让光芒离开它们的视线，并会为保护光芒不惜生命。如果被强行从光芒身边带走，生物会试着自己找到回到它们身边的路。生物无法处于这种状态下进行长休或短休。在<span lang="EN-US">24</span>小时后，被魅惑的生物如果能看到光芒，则必须再次进行豁免，成功则结束该效应。如果光芒在这时不在其视线内，则该生物豁免自动成功。

每一团诱饵光具有<span lang="EN-US">AC17,100</span>生命值，并免疫暗蚀与毒素伤害。

防护<span lang="EN-US">Abjuration</span>

充满防护魔法的风暴会斥退船只，仿佛在保护海中的什么东西。在风暴结束之前，其中的一艘船被向随机方向传送<span lang="EN-US">20</span>英里。

咒法<span lang="EN-US">Conjuration</span>

咒法魔法在风暴的中心搅动时，狂风和暴雨的生灵会从遥远的领域出现，攻击它们所遇到的一切事物。船只会遭遇<span lang="EN-US">1d3</span>只敌对的**气元素<span lang="EN-US">air elementals</span>**。

预言<span lang="EN-US">Divination</span>

当预言魔法渗入风暴时，呼啸的风和残余的低语会扰乱船员们的所知；在接下来的<span lang="EN-US">1d3</span>天内，船员的所有素质值检定获得劣势。

惑控<span lang="EN-US">Enchantment</span>

充满惑控魔法的风暴会消除生物的危机感，吸引水手并使他们加速驶入危险之中。在风暴期间，为解决危机或管理船只所进行的任何检定都是劣势。

塑能<span lang="EN-US">Evocation</span>

这些凶猛的风暴被雷电、大片酸雨、炸裂的流星和其他危险所包裹。在风暴之中的每天之后，船上的每一个部件都会受到<span lang="EN-US">1d10</span>火焰、<span lang="EN-US">1d10</span>强酸以及<span lang="EN-US">1d10</span>闪电伤害。另外，投一个<span lang="EN-US">d20</span>并加上船员的素质，在<span lang="EN-US">10</span>以下的场合，船员的素质降低<span lang="EN-US">1</span>，并杀死其中<span lang="EN-US">1d6</span>个船员。

幻术<span lang="EN-US">Illusion</span>

一次充满幻术的风暴不会对船只造成直接伤害，但经验丰富的航海家直到它会构成一种潜在威胁。在风暴中，船只朝着一个并非它原本预定航向的随机方向航行。

死灵<span lang="EN-US">Necromancy</span>

幽魂在风暴中号哭和旋转，而那些死去已久的水手遗骸则从他们潮湿的坟墓中站起。在风暴期间，<span lang="EN-US">1d4</span>只**幽灵<span lang="EN-US">specters</span>**，<span lang="EN-US">2d4</span>只**食尸鬼<span lang="EN-US">ghouls</span>**和<span lang="EN-US">4d6</span>只**僵尸<span lang="EN-US">zombie</span>**从海浪之中浮起并攻击船只。

变化<span lang="EN-US">Transmutation</span>

这些剧烈而难以预料的风暴导致海浪扭曲。风暴将船只周围的水变成冰、石头和奇怪物质的混合物，导致航行变得困难并损毁船只。船体受到<span lang="EN-US">4d10</span>钝击伤害，在风暴中船的速度下降一半。

<span lang="EN-US"></span>

沙洲<span lang="EN-US">SANDBARS</span>

海浪和洋流在靠近海岸线的浅水中让沙子沉积起来。这些沙洲形成了一条长度和宽度从<span lang="EN-US">10</span>尺带<span lang="EN-US">3</span>英里之间的线。涨潮时，沙洲位于海面以下，而在落潮时它们则会从水面上浮现出来。

<span lang="EN-US"></span>

渡过沙洲<span lang="EN-US">NAVIGATING SANDBARS</span>

沙洲对船只形成危险的阻碍。尤其是涨潮时，它们并不太明显。向海浪下面的沙洲移动的船只的船员必须具有<span lang="EN-US">12</span>及更高的被动差距，否则不能看到航向上的沙洲（见“注意威胁<span lang="EN-US">Noticing Threats</span>”，<span lang="EN-US">200</span>页）。当一艘船经过一个被淹没的沙洲时，沙洲视作是困难地形，船只必须成功通过一次敏捷豁免。这次豁免<span lang="EN-US">DC</span>与沙洲的深度所对应，如沙洲<span lang="EN-US">DC</span>表所示。如果船只没能通过这一豁免，它就会陷在沙丘上。通过使用一个动作进行力量检定（加值等于船员的素质值），来对抗沙洲的<span lang="EN-US">DC</span>，船只能够从中脱困。

腐蚀雪<span lang="EN-US">SAPPING SNOW</span>

腐蚀雪——这是海洋探险者们称呼覆盖海底的死亡生物粉末状残骸的说法。当这种物质被死灵魔法充斥时，它就会变成一种吸食生命的碎屑。

当一个生物在其回合开始时接触这些腐蚀雪时，这一生物必须成功通过<span lang="EN-US">DC15</span>的体质豁免检定，否则受到<span lang="EN-US">10</span>（<span lang="EN-US">3d6</span>）暗蚀伤害。该生物的生命值上限减少受到暗蚀伤害的数值。这一减少持续到生物完成一次长休。如果这一效果将生物的生命值上限降低到<span lang="EN-US">0</span>，则该生物立即死亡，它的身体瓦解成腐蚀雪。

漩涡中的船只<span lang="EN-US">VESSELS IN WHIRLPOOLS</span>

团队检定代表五分钟内的工作。这次检定的<span lang="EN-US">DC</span>从漩涡等级表格中选择或者投掷一个<span lang="EN-US">d4</span>随机决定。船长、大副、水手长以及舵手每人进行一次属性检定，如漩涡检定表所示。如果没有人为特定重要成员进行检定，将会导致团队检定失败。同时，为船员投掷一个<span lang="EN-US">d20</span>骰，使用其素质值作为该次掷骰的调整值，并与检定的<span lang="EN-US">DC</span>相比较。

用团队的检定中成功的数量扣除失败的数量——包括重要成员和船员的——然后查阅漩涡检定结果表。
