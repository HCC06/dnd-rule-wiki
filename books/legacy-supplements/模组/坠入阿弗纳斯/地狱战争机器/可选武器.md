<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">一种车载武器可以换成另外一种，但</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">加装比通常情况下更多的武器。此处列举了一些可选武器。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">酸液喷枪</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">Acidic Bile Sprayer</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">这个武器用软管连接着能产生酸液的恶魔脏器，用以腐蚀目标的血肉。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">酸液喷枪（需要一名成员并提供半掩护，充能5-6）</span>**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">。</span>***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">酸液喷枪的射程为30尺锥形。在范围内的生物需要通过一个DC12的敏捷豁免，失败则受到40（9D8）点强酸伤害，成功则伤害减半。因此攻击而HP降为0的生物会被溶解，所有携带的物品会被留在原地</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">火焰喷射器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">Flamethrower</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">尽管对恶魔无效，这种武器在应对可燃生物和物体时很有效。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">火焰喷射器（需要一名成员并提供半掩护）</span>**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">。</span>***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">火焰喷射器的攻击范围为5尺宽的60尺直线 。范围内的生物需要通过一次DC15的敏捷豁免，否则受到18（4D8）点火焰伤害，成功则伤害减半。火焰会点燃范围内所有未被装备或携带的可燃物。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱尖嚎装置</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">Infernal Screamer</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">这个武器台上装着一个由融化的蜡制成的扭曲人形躯干，</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; BACKGROUND: rgb(252,252,254); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-shading: rgb(252,252,254)">在它的肩胛骨之间有一个带刺的曲柄。当曲柄转动时，这个人物发出痛苦的心灵尖叫。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; BACKGROUND: rgb(252,252,254); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-shading: rgb(252,252,254)"></span>

***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">地狱尖嚎装置（需要一名成员并提供半掩护）</span>**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">。</span>***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">选择一个120尺内的可见生物为目标，目标生物需要通过一个DC15的感知豁免，否则受到26（4D12）点心灵伤害，如果豁免成功，则伤害减半。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">冥河水喷枪</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">Styx Sprayer</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">喷枪的软管连接着装有30加仑冥河水的水箱。每次攻击消耗10加仑水。需要通过冥河才能给水箱加水。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">冥河水喷枪（需要一名成员并提供半掩护：3次）</span>**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-ansi-font-weight: bold">。</span>***<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">远程法术攻击：+5命中，30尺射程，单个目标。命中时：生物被冥河水命中，且受到</span>*<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">弱智术</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; FONT-STYLE: italic; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">Feeblemind</span>*

效果（豁免DC20）。已经受到弱智术效果的对此效果免疫，因为它已经习惯冥河水了。如果目标豁免失败，且此效果在30天内未被解除，则<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">此效果会永久生效，并且目标会失去所有记忆，变成一个近乎于无意识的躯壳。这种情况下，只有神灵的干预才能消除这种影响。</span>
