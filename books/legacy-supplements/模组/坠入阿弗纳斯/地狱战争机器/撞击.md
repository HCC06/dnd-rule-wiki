**<span style="FONT-SIZE: 14pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'; mso-bidi-font-family: 'Times New Roman'; mso-font-kerning: 1.0000pt">撞击</span><span style="FONT-SIZE: 14pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'; mso-bidi-font-family: 'Times New Roman'; mso-font-kerning: 1.0000pt">Crashing</span><span style="FONT-SIZE: 14pt; FONT-FAMILY: 'Times New Roman'; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-fareast-font-family: 黑体"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">当</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">撞上有可能对其造成伤害的物体时，比如一面铁墙，或是和它一样大小或是比它更大的车辆时，战车会停下，并且在撞击前的一回合每移动10尺就会受到1D6点钝击伤害（最大20D6）。被战车撞击的物体也会受到相同的伤害。如果此伤害低于</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">的伤害阈值，那么此次撞击不造成任何伤害。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">无论战车有没有在撞击中受到伤害，每个战车内的生物都必须通过一次DC15的力量豁免，否则在撞击前的一回合，战车每移动10尺就会对车内生物造成1D6点钝击伤害（最大20D6），如果豁免成功，则受到一半的伤害。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">撞击生物</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">Crashing into Creatures</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">通过闯越生物占据的空间，</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">可以冲撞生物。生物可以通过一个DC10的敏捷豁免来躲开，并免受伤害。如果豁免失败，那么战车在撞击前的一回合每移动10尺就会对生物造成1D6点钝击伤害（最大20D6）。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">如果</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">的体型比所撞的生物大两个级别或以上，那么在撞击之后它可以继续移动。否则战车会停止，并且在撞击前的一回合，战车每移动10尺就会对车内生物造成1D6点钝击伤害（最大20D6），如果豁免成功，则受到一半的伤害。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>
