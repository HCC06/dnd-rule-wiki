**驾驶室Helm**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">当驾驶员处于失能状态，或离开驾驶室，或不做任何动作来改变战车的移动路线和速度，那么战车会按照驾驶员上一回合所设定的速度和方向前进，直到撞上足够令它停止的障碍物为止。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: Wingdings; mso-bidi-font-family: 宋体; mso-font-kerning: 1.0000pt; mso-fareast-font-family: 宋体"><span style="mso-list: Ignore">l </span></span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">启动或停止</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">的引擎。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: Wingdings; mso-bidi-font-family: 宋体; mso-font-kerning: 1.0000pt; mso-fareast-font-family: 宋体"><span style="mso-list: Ignore">l </span></span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">在引擎已经启动的前提下，让战车使用疾走或撤离动作。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">武器操作台</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">Weapon Stations</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">的每个武器都会占据一个操作台。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">战车上的武器可以被替换成不同种类（见“武器替换”），前提是操作两种武器所需要的成员要求相同。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'; mso-bidi-font-family: 'Times New Roman'; mso-font-kerning: 1.0000pt">灵魂燃料</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'; mso-bidi-font-family: 'Times New Roman'; mso-font-kerning: 1.0000pt">Soul Fuel</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 'Times New Roman'; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt; mso-fareast-font-family: 黑体"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">每辆</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">的引擎核心都是一个用灵魂币作为燃料的熔炉。战车的驾驶室内由个窄槽，可以从此处放入灵魂币。放入此槽的灵魂币会进入熔炉，熔炉的大小比车辆的大小要小两级。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">地狱战争机器</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">的熔炉会立即消耗掉灵魂币，并同时一次性消耗掉灵魂币剩余的充能，在此过程中，灵魂币会被摧毁。原本困在灵魂币内的灵魂会被困在熔炉内，根据灵魂币被消耗时剩余的充能来给引擎提供相应时间的能量：1次充能，24小时；2次充能，48小时；3次充能，72小时。如果在给引擎的充能时间结束后，灵魂依旧被困在熔炉内，那么它会被摧毁。此情形下，即使是神圣干预也无法救回被摧毁的灵魂。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">无论其大小，熔炉都可以容纳任意数量的灵魂，它们痛苦的悲鸣声可以在60尺外被听见。每个被投入熔炉的新灵魂都会给战车提供燃料，让它们运行更长的时间。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">往战车的熔炉倒入一瓶恶魔秽液可以为战车提供1分钟额30尺移速加成。在此状态下，每当驾驶员的回合开始时，投一个D20。如果结果为1，那么战车会遭受厄运表（P220）上，熔炉破裂的情况。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>
