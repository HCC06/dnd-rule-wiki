**<span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'; mso-bidi-font-family: 'Times New Roman'; mso-font-kerning: 1.0000pt">达成协议 Entering into a Deal</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 黑体; FONT-WEIGHT: bold; COLOR: rgb(128,0,0); mso-ascii-font-family: 'Times New Roman'; mso-hansi-font-family: 'Times New Roman'; mso-spacerun: 'yes'; mso-bidi-font-family: 'Times New Roman'; mso-font-kerning: 1.0000pt"></span>**

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">尽管魔鬼渴望将凡人的灵魂困在有这些有约束力的契约中，但它们只能在特定情况下达成交易。最简单的方法就是在九狱里达成交易，在九狱里，魔鬼更接近其地狱等级制度，更容易引导自己的力量。在多元宇宙的其他地方，大多数魔鬼离他们的权力结构太远，在没有特殊情况下无法达成有约束力的交易。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">交易包括对涵盖各方预期条款的提议，以及一份封缄交易并令其具有约束力的契约。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">高阶和低阶魔鬼不能在家园位面外自由达成交易，但也并非毫无办法。如果在另一个位面的召唤者知道这个魔鬼的真名或有它的邪符（如《怪物图鉴》中的“魔鬼真名和邪符”中所述），与其将魔鬼封困住以提供服务，召唤者可以创建一个允许魔鬼达成交易的仪式法阵。一旦交易达成，魔鬼就会回到九狱。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>

<span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt">一个想要和角色做交易的大魔鬼可以亲自出面，也可以派一个仆从来提议交易并代表它谈判。一个角色也可以通过杀死一个混乱或善良阵营的类人生物，并向大魔鬼祈祷一小时来联系大魔鬼。大魔鬼听到了角色的祈祷，可以选择在梦中或幻象中做出回应，也可以派出一个有权代表他做交易的仆从出面。</span><span style="FONT-SIZE: 12pt; FONT-FAMILY: 宋体; COLOR: rgb(0,0,0); mso-spacerun: 'yes'; mso-font-kerning: 1.0000pt"></span>
