**<span style="FONT-FAMILY: 宋体; mso-ascii-font-family: Calibri; mso-hansi-font-family: Calibri">附赠熟练项</span><span lang="EN-US" style="FONT-FAMILY: &quot;Palatino Linotype&quot;,serif; mso-bidi-font-family: &quot;Palatino Linotype&quot;">Bonus Proficiency  
</span>**<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

在第1级选择该领域时，你获得重甲的熟练项。

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

**守望相助Solidarity's Action**  

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

第1级起，当你执行协助动作协助友方生物进行攻击时，你可以以一个附赠动作发起一次武器攻击。你可以使用该特性的次数等于你的感知调整值（至少为一次）。在完成一次长休后重新恢复所有的使用次数。

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

**引导神力：维持生命Channel Divinity: Preserve Life**  

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

第2级起，你可以用引导神力特性来恢复严重的伤害。  
你用一个动作展示圣徽，引导治疗能量来恢复等于你牧师等级五倍的生命值。你指定身边30尺内的任意生物作为该特性的目标，再为其分配从中获得的治疗能量。该特性不能将目标恢复到其生命值上限的一半以上。而且你不能对亡灵或构装生物使用该特性。

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

**引导神力：欧柯塔之佑Channel Divinity: Oketra's Blessing  
**

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

第6级起，当一个30尺内你可见的生物进行攻击检定时，你可以花费一个反应使用引导神力令该次检定获得+10加值。你可以在看到掷骰结果后DM公布是否命中前，决定是否使用该特性。

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

**神圣打击 Divine Strike  
**

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

第8级起，你可以使你的武器充满神圣能量。你在每个自己回合里有一次机会，可以在用武器攻击命中某生物时，使该攻击额外造成1d8点与该武器伤害类型相同的伤害。14级时，该伤害提升到2d8。

<span style="FONT-FAMILY: 宋体; FONT-WEIGHT: normal; COLOR: windowtext; mso-bidi-font-weight: bold; mso-bidi-font-family: 宋体"><span lang="EN-US"></span></span>

**极效治疗Supreme Healing**  
第17级起，当你需要用一个法术来掷一粒或多粒骰子计算你所恢复的生命值时，所有骰子将取其最高值。例如，法术为某一生物恢复2d6生命值，则其结果直接取12。
