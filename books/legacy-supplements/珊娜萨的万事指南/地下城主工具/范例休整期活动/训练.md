如果有足够的空闲时间和教练的指导，一个角色可以学习一项语言或者是选择一套工具获得熟练。

<span lang="EN-US"></span>

资源

学习一种语言或工具需要至少<span lang="EN-US">10</span>个工作周训练，不过这个时间可以被人物的智力调整值缩短，每点<span lang="EN-US">1</span>工作周（智力上的罚值不会增加所需时间）。训练同时也需要花费<span lang="EN-US">100gp</span>每工作周的金钱。

<span lang="EN-US"></span>

难题

在训练中产生的难题往往与教师有关。角色进行训练的每个休整期都有<span lang="EN-US">10%</span>的机会引发难题，训练难题如表中示例。

<span lang="EN-US"></span>

表：训练难题

|                                                                     |                                                                                                                     |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| <span lang="EN-US" style="FONT-SIZE: 11pt; COLOR: black">d6</span>  | <span style="FONT-SIZE: 11pt; COLOR: black">难题</span>                                                             |
| <span lang="EN-US" style="FONT-SIZE: 11pt; COLOR: black">1\*</span> | <span style="FONT-SIZE: 11pt; COLOR: black">你的教练失踪了，你不得不花费一周去重新找一个。</span>                   |
| <span lang="EN-US" style="FONT-SIZE: 11pt; COLOR: black">2</span>   | <span style="FONT-SIZE: 11pt; COLOR: black">你的教练用一种罕见的、古老的方法训练你，这引来了别人的指指点点。</span> |
| <span lang="EN-US" style="FONT-SIZE: 11pt; COLOR: black">3\*</span> | <span style="FONT-SIZE: 11pt; COLOR: black">你的教练是被派来刺探你近期计划的间谍。</span>                           |
| <span lang="EN-US" style="FONT-SIZE: 11pt; COLOR: black">4</span>   | <span style="FONT-SIZE: 11pt; COLOR: black">你的教练是通缉犯。</span>                                               |

\*可能涉及对手
