# 拟身怪 Mimic 伪装成不起眼物件的变形异兽
**栖息地：** 幽暗地域，城市；**宝藏：**任意
拟身怪在平常形态下如同游荡的胃袋，圆鼓鼓的身体上布满异星眼眸和牙齿。它们能够改变颜色、纹理、尺寸，以复制与它们大小相近的静物。拟身怪利用外形作为伪装和诱饵。一旦受害者接近，拟身怪就会发起攻击，用满是粘液的伪足和利齿猛烈攻击。吃掉猎物以后，拟身怪就会换个位置，改变形态，等待下一顿餐食。 使用以下列表作为拟身怪外形的灵感。 祭坛
