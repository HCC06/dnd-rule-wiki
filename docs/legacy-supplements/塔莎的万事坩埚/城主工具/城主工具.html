城主工具

# **第四章：城主工具 Dungeon Master’s Tools ** 准备和运作D&D战役时，地下城主会用到许多种工具。作为DM，你的工具包含了你的想象力、对玩家兴趣所在的洞察力、敏锐的叙事嗅觉、幽默感、倾听能力、运用规则的能力，等等。本章将为你的工具箱增添指引和为多种情况准备的可选规则。本章同样包含了一个章节的预设谜题，可供放进任何战役。 文中的工具基于《城主指南》与《怪物手册》中的内容。你可以全部使用、部分使用、或是完全不用这些工具，也可以自由调整它们的运转方式。你的团体能够玩的开心是最为重要的，所以，把这些规则收为己用，令它们符合你团队的口味吧。
