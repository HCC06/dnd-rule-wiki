牧师（TCE）

牧师 Cleric
牧师职业在本节中获得了新的职业特性与子职业选项。

可选职业特性 Optional Class Features
你可以在达到特定等级时获得玩家手册中描述的职业特性。这一部分提供了一些你作为牧师的额外特性。与玩家手册中的特性不同的是，你不会自动获得这些特性。在你达到该部分描述的职业特性的需求等级且决定获得这些特性时，与你的 DM先商量商量。这些特性能够被一个一个单独选走，这意味着你可以使用其中一些、全部或不使用它们。
若你将一个特性替换为另一个特性，那么你将不会受到被替换的特性所带来的好处，你也不会满足在游戏中任何需求被替换的特性的东西。

神力掌控 Harness Divine Power
第2级牧师特性
你可以将一次引导神力次数转化为你法术的魔力。以一个附赠动作，你触碰你的圣徽进行祈祷，以此，你恢复一枚环阶 不超过你熟练加值的一半（向上取整） 的法术位。
你能使用这一特性一次，当你在此职业上达到更高等级后你可以使用更多次：6级后两次；18级三次。
在你完成一次长休后，你恢复所有已消耗的使用次数。

替换戏法 Cantrip Versatility
第4级牧师特性
当你在此职业获得了属性值提升时，你可以将一个你通过此职业的施法特性学会的牧师戏法替换为牧师法术列表中的另一个戏法。

受祝击 Blessed Strikes
第8级牧师特性，替代 神圣打击Divine Strike 特性或 强力施法Potent Spellcasting 特性
在战斗中，你受到神力的祝福。当你通过戏法或武器攻击来伤害一个生物时，你能够对该生物造成额外1d8光耀伤害。当你造成一次此伤害后，你直到自己的下一个回合开始前无法再次使用此特性。
