游侠（TCE）

游侠 Ranger
游侠在这一部分得到了一些新的特性与子职。

可选职业特性 Optional Class Features
你可以在达到特定等级时获得玩家手册中描述的职业特性。这一部分提供了一些你作为游侠的额外特性。与玩家手册中的特性不同的是，你不会自动获得这些特性。在你达到该部分描述的职业特性的需求等级且决定获得这些特性时，与你的DM先商量商量。这些特性能够被一个一个单独选走，这意味着你可以使用其中一些、全部或不使用它们。
若你将一个特性替换为另一个特性，那么你将不会受到被替换的特性所带来的好处，你也不会满足在游戏中任何需求被替换的特性的东西。

熟练探险家 Deft Explorer
第1级游侠特性，替换 自然探索者Natural Explorer 特性
 你是一个无与伦比的探险家和求生专家。你在1级时获得对应的特性，然后在第6级和第10级时再各获得对应的特性。

精明 Canny（1级）
你选择一项你拥有熟练的技能，你在使用该技能时所做的任何属性检定中的熟练加值变为两倍。
你还可以额外说、读、写两门额外的自选语言。

越野 Roving（6级）
你的步行速度增加5。同时，你获得等于你步行速度的攀爬速度和游泳速度。

不知疲倦 Tireless（10级）
以一个动作，你能够给予自己1d8+你的感知调整值的临时生命值（至少1点临时生命值）。你能够使用此特性的次数等于你的熟练加值次，当你完成一次长休后，你恢复已消费的全部次数。
此外，当你完成一次短休时，你的力竭等级减少1级（若有）。

夙敌 Favored Foe
第1级游侠特性，替换 宿敌Favored Enemy 特性并能对 屠灭众敌Foe Slayer 特性起效
当你以一次攻击掷骰命中一个生物时，你可以呼唤你与自然的神秘羁连以将目标标记为你的夙敌，持续一分钟或直到你失去专注(如同你在专注一道法术)。
你在你的回合内第一次命中你的夙敌并对其造成伤害，包括你标记它的时候，你可以造成额外的1d4点伤害。你可以使用同等于你的熟练加值次该特性来标记你的夙敌，你在完成一次长休后恢复所有已使用的次数。这个特性的额外伤害在你达到特定等级时会增加:6级时1d6，14级时1d8。

战斗风格选项 Fighting Style Options
第2级游侠特性
当你在选择战斗风格时，下列选项加入你的可选战斗风格中。

盲斗Blind Fighting
你获得10尺的盲视感官。在此范围内，你可以切实的看见非 完全身掩护total cover 的任何东西，即使你处于 目盲Blinded 状态或是身处黑暗之中。不仅如此，你还可以看见此范围内的 隐形Invisible 生物（除非对你而言该生物成功躲藏了起来）。

德鲁伊教战士Druidic Warrior
你学会两个来自德鲁伊法术列表的你自选的戏法。它们对你视作游侠法术，感知是你施展他们的施法属性。当你升级时，你可以将其中一个戏法替换为德鲁伊法术列表中的另一个戏法。

投掷武器战斗Thrown Weapon Fighting
你可以在使用一件带有投掷属性武器进行进攻时，将那件武器作为攻击的一部分而拔出。另外，在你以一件投掷武器于远程攻击中命中对手时，你的伤害掷骰+2。

法器 Spellcasting Focus
第2级游侠特性
你可以使用一件德鲁伊法器作为你的游侠法术的法器。一件德鲁伊法器可能会是一枝槲寄生或者冬青木的枝条，用紫衫木或者其他特殊木材制作而成的魔杖或权杖，一柄从活化树木上取来的法杖，或是一件混合了神圣动物的羽毛、毛发、骨头与牙齿的物件。

原始意识 Primal Awareness
第3级游侠特性，替换 原初意识Primeval Awareness 特性
你可以通过与自然的共鸣而集中你的意识:若你尚未习得下方表格种的法术，那么你在抵达对应等级时学会下方原始意识法术表格中的法术。这些法术不占用你的游侠已知法术。

替换武术 Matrial Versatility
第4级游侠特性
当你的游侠职业等级抵达能让你获得属性值提升职业特性时，你能用一个游侠可用的战斗风格替换原有的一个战斗风格。这种变化代表着你改变了武术训练和练习的重点。

自然面纱 Nature's Veil
第10级游侠特性，替换 遁术Hide in Plain Sight 特性
你可以暂时地使用自然的力量将你的身形遮蔽。以一个附赠动作，你可以魔法地使自己与正携带、着装的任何装备变得隐形，直到你的下个回合开始。

你可以使用该特性的次数同等于你的熟练加值次，完成一次长休后，你重获全部已消耗的使用次数。
