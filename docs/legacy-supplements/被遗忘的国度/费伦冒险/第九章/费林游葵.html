费林游葵

费林游葵 Phaerimm Agent

费林游葵在地表世界四处游走，搜寻强大的魔法以统治他人并解救被囚禁的同族。

AC 14 先攻 +4（14）
HP 123 （19d10 + 19）
速度 10尺，飞行40尺（悬浮）

特质Traits

魔法抗性Magic Resistance。费林游葵对抗法术和其他魔法效应时进行的豁免检定具有优势。

动作Actions

多重攻击Multiattack。费林游葵使用恐怖毒刺或扭心射线发动共计3次攻击。
恐怖毒刺Dread Stinger。近战攻击检定：+7，触及5尺。命中：7（1d6+4） 穿刺伤害外加13（2d12） 点毒素伤害。
扭心射线Mindwarp Ray。远程攻击检定：+7，射程120尺。命中：13（2d8+4） 心灵伤害，目标陷入魅惑状态直至费林游葵下个回合开始。
施法Spellcasting。费林游葵施展以下一道法术，无需任何法术成分并使用魅力作为施法属性（法术豁免DC15）：
随意：侦测魔法Detect Magic，侦测思想Detect Thoughts，法师之手Mage Hand
每项1/日：鹰眼术Clairvoyance，高级幻影Major Image，突触静止Synaptic Static

附赠动作Bonus Actions

传送Teleport。费林游葵传送至30尺内一个其可以看见的未被占据空间。

反应Reactions

护身魔法Protective Magic（3/日）。费林游葵施展法术反制Counterspell或护盾术Shield（触发条件见这些法术），使用与施法动作相同的施法属性。
