堕落女王

身覆锈蚀的海盗女王

锈蚀使我变得强大，赐予我长寿，还保护我免受敌人侵害。而你们竟称它为诅咒！这或许是对我敌人的诅咒。但对我来说，却是恩赐！

——堕落女王

若要说月影群岛全境威名远扬的超自然锈蚀咒受害者都有哪些，那为首的当属被称为堕落女王的海盗女王。她驾驶着月影群岛规模最大、防御最为坚固的战船在海上横行，这艘船是她舰队中的珍宝。堕落女王用锈迹斑斑的金属板加固船身，船上满载一群嗜血成性的锈蚀狂战士Rusted berserkers （详见本章）。
堕落女王是一位强壮的人类女性，有着一头红色的长发。尽管锈蚀咒已将她的大部分四肢变成了金属，但她仍抵御住了诅咒那通常会给受害者带来弱化的侵蚀。她的船员们坚称，是那对掠夺财宝永不满足的渴望使她保持强健。

历史 History

鲜有人知晓堕落女王的过往。她更愿将精力倾注于大胆的冒险，以月影群岛最负盛名的海盗女王之名行事。但堕落女王其实是雷娜塔Renauta的称号，她是欧夫高特Olfgaut之女，也是诺兰领主欧夫斯文Olfsvenn的妹妹。雷娜塔得知了锈蚀咒的存在，并将其视为通往力量与勇武之路接受了该诅咒。
堕落女王与她的兄长保持着秘密联系，二人偶尔会互相分享有用的情报。欧夫斯文希望自己能引导堕落女王回归本名雷娜塔这一身份与角色，但堕落女王却更钟情于自己当下的新身份。

使用堕落女王 Using Queen Forfallen

堕落女王是一位实力强劲的战士与领袖。她麾下那些由锈蚀生物与海盗组成的队伍在月影群岛声名远扬，因此，在以她为主线的任何战役中，角色们应尽早知晓她的名号。他们甚至有可能远远望见她立于一艘嗜血劫掠者战船的船首，身后留下一片惨遭蹂躏的废墟。
堕落女王与诺兰领主欧夫斯文之间的秘密关系或许会驱使这位领主做出看似保护或庇护这位凶残海盗女王的举动。在原本单纯的追捕海盗行动中揭开堕落女王的身世之谜，能够为调查、权谋博弈以及家族纷争带来意想不到的契机。

个性 Personality

堕落女王的个性如同她那被金属包裹的肌肉般强硬有力。她嗓音洪亮，在船与船的激战中也能传递信息；她态度坚决、直率坦诚，令潜在的盟友难以抗拒与她结盟。她不断壮大的支持者网络印证了她出色的外交手腕，以及在果断举止背后那精明的头脑。
她与那些游离于诺兰社会之外的恶棍、强盗、走私犯等群体攀谈，许以钱财与荣耀。更阴险的是，她宣扬锈蚀咒是通往力量的道路，而非导致衰弱的灾祸，那些渴望获得她这般力量的人便主动寻求感染。在月影群岛，堕落女王为传播这一诅咒所做的贡献远超其他人。
和所有感染锈蚀咒的受害者一样，堕落女王极度渴望破坏自然。不过她尚能克制这种欲望，转而将精力投入到与其他海盗以及同样感染锈蚀咒之人建立联盟上。但对于砍伐原始森林以获取造船木材，或是从野外山丘中挖掘矿石，她都毫无愧疚之感。
