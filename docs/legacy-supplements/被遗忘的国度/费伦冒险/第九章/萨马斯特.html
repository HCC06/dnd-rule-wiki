萨马斯特

萨马斯特 Sammaster

奠基龙巫秘教的巫妖

萨马斯特是龙巫教的建立者与前任领袖——这一邪恶组织致力于将活着的巨龙转化为摧残世界的亡灵恶孽。萨马斯特曾在数个世纪前被击败，而如今的他已化作巫妖之身，其本源之中还蕴藏着更强大的龙巫妖之型。

历史 History

萨马斯特原是生于数个世纪前的聪慧法师。学徒时代，天资聪颖的萨马斯特曾云游四方，钻研与巨龙相关的魔法。在从伊尔明斯特和艾拉斯卓·银手等大法师门下出师后，萨马斯特便浸淫于死灵和惑控魔法之中。他创造了能将巨龙转化为龙巫妖的手段，魔法性地令巨龙为其服务，甚至知晓如何令全费伦的龙陷入狂怒。他认为自己的智识早然超越了道德，逐渐变得偏执、怨恨、狂妄。
萨马斯特在旅行中结识了那些将巨龙视作原初伟力之显化的人。他将自己妄图支配龙族的欲望伪装成对龙族的忠诚，将这些龙之狂信者们统合在自己的麾下。这支新生教团被萨马斯特命名为龙巫教。萨马斯特利用教众对他的死灵知识的痴迷，向教众灌输道：龙巫妖终将统治费伦。他的教团投身于帮助恶龙成为龙巫妖，致力于令其所期许的邪恶时代化作现实。
萨马斯特最终死在了战场上，但他早已为自己的败亡准备了后手。他利用龙巫妖扭曲的复生形式，将自己也化作一名巫妖，从而在死后继续于幕后领导龙巫教，开发支配并腐化龙族的新手段。

使用萨马斯特 Using Sammaster

萨马斯特可在一场着眼于龙巫教的战役中作为可怕的反派登场。他常常游历于国度诸地，视察龙巫教基层组织，这为玩家提供了与这尊巫妖互动的机会——即使他们正处在低等级的冒险中。萨马斯特的现身意味着看似小股的邪教行动不过是其背后巨大阴谋的只鳞片爪。
与萨马斯特的直接对决可作为一场战役的史诗高潮。然而萨马斯特以难于被杀而闻名，仅仅杀死他一次不过是令其回归灵魂宝石，从中复活，继续荼毒费伦；角色们必须设法摧毁萨马斯特的灵魂宝石才能将其彻底击败。

个性 Personality

萨马斯特长相英俊迷人，但他难以掩盖自己对亡灵巨龙长久以来的病态痴迷。只有龙巫教的内环才是他真正的盟友——同时也臣服于他的命令。萨马斯特很少于一地久留，因为他认为只有自己亲自出马才能有效地监管整个国度中的龙巫教活动。教团中最重要的那些阴谋全都要由萨马斯特亲自察看，他的突然视察经常令手下心惊。萨马斯特不像其它巫妖那样将灵魂藏于命匣之内，而是像龙巫妖一般将其封于一枚灵魂宝石中。如果摧毁萨马斯特的巫妖之躯，它将复活为一尊以其灵魂宝石为核心的龙巫妖。唯有将这一龙巫妖形态也打败，并赶在灵魂宝石将他重塑为巫妖之躯前把宝石摧毁，才能将萨马斯特彻底消灭。那些在日后被尊为英雄的人曾多次以为自己战胜了萨马斯特，但这位聪明绝顶的法师总能归来，领导着龙巫教深入更绝望、更罪恶的阴谋之中。
