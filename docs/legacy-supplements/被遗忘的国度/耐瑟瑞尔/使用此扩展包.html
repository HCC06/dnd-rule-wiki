使用此扩展包

使用此扩展包 Using This Supplement

本扩展包包含了你在古代耐瑟瑞尔——被遗忘的国度世设中一个早已陨落的帝国——开展《龙与地下城》冒险所需的全部工具。文中引用的怪物数据均出自《怪物图鉴》或附录B。
这段引言概述了古代耐瑟瑞尔及其历史。虽然本引言适合所有玩家阅读，但本扩展的其余内容仅供地下城主使用。
本扩展包含以下内容：
第一章提供了关于在古代耐瑟瑞尔中展开冒险的资讯，包括浮空城伊莱纳Eileanar及螺湾城Conch的详细介绍。同时也包含设定在这些城市中的地图与冒险工具。
第二章阐述了古代耐瑟瑞尔因魔法滥用与开采而常见的环境效应与灾害。
第三章详述了当代的时光之门，角色可借此前往古代耐瑟瑞尔，并附上了相关地图与设定在这些地点附近的冒险。
第四章提供了设定在古代耐瑟瑞尔中的地图与冒险。
附录 A收录了角色可能在冒险中获得的耐瑟瑞尔魔法物品。
附录 B提供了古代耐瑟瑞尔中可见生物的怪物图鉴。

确保所有人玩得开心 Ensuring Fun for All

本扩展涉及的主题中，有些可能会让部分玩家感到敏感或不适。在你将本扩展纳入游戏之前，请仔细阅读其中内容，回顾《城主指南》中"确保所有人玩得开心"一节，并务必按照"硬底线与软底线"一节中的说明，与所有玩家进行沟通。
