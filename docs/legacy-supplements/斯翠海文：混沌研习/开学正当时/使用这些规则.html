使用这些规则

使用这些规则 Using These Rules

本书中的冒险也包括了社交遭遇，也就是角色可以和其他学生 NPC 们（相应的 NPC 将在下文“同学”章节提到）进行互动的事件。当这样的遭遇开始时，可以使用本书的冒险中提到的朗读文本来描述这个场景，然后让角色们逐一和学生 NPC 进行互动。
每个和学生 NPC 在社交遭遇中进行了互动的玩家角色，都可以获得一个或更多的和该 NPC 之间的关系点数。
和特定 NPC 之间的关系点数量决定了玩家角色和该 NPC的关系。
