伯劳舰

伯劳鸟舰 Shrike Ship

这种迅捷的舰船是一种相对现代的设计，一经推出很快就受到商人和海盗的欢迎。 伯劳鸟舰的腿使她能够安全地降落在地面上。 这艘舰船可以漂浮，但不是为在水上航行而建造的，并且在波涛汹涌的大海中会迅速地下沉。
伯劳鸟舰上的标准武器包括三门弩炮一个在艏楼，一个在顶层甲板的中间，一个在舰船尾。 危急情况下，该舰的加固船首可用作穿刺撞角。

伯劳鸟舰概要 Shrike Ship Summary：
AC：15（木质）
载货量 Cargo：20 吨
HP: 250
船员 Crew：11
伤害阈值 Damage Threshold：15
龙骨/横梁 Keel/Beam：100 尺/20 尺。
速度 Speed：飞行 70 尺（8 里/小时）
价格 Cost：20000gp

3 门弩炮 Ballistae（每门船员：3）
AC：15
HP：50
价格：50gp（弩炮），5gp（弩箭）
弩炮需要 1 个动作装填，1 个动作瞄准，1 个动作发射。
弩箭 Bolt。远程武器攻击：+6 命中，射程 120/480 尺，一个目标。 命中：16 (3d10) 点穿刺伤害。

穿刺撞角 Piercing Ram
AC：-
HP：-
价格：-（包含在舰船的价格中）
当舰船撞击另一个物体或一个超巨型的生物时，舰船的法驱者可以进行以下攻击（参见本章前面的“碰撞”部分）。
穿刺撞角 Piercing Ram。近战武器攻击：+8 命中，触及 0 尺，一个物体或超巨型生物。 若命中：88 (16d10) 点穿刺伤害。 伯劳鸟舰受到一半的伤害并急停。 未命中：攻击不造成伤害，目标移动到最近的未被占据且不在伯劳鸟舰路径上的空间，伯劳鸟舰可以继续移动，如果她还有任何移动。
