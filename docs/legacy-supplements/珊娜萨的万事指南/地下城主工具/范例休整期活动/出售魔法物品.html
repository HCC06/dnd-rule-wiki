出售魔法物品

出售魔法物品绝非易事。骗子和小偷总是在寻找一个容易得手的目标。即使角色们找到了一个合法的买家，他们也不一定能得到理想的报价。 资源 为你的魔法物品寻找一个买家需要一个工作周的时间和100gp的花费，用于推广销售。同一时间你只能选择一件物品出售。 决议 想要出售魔法物品的角色必须进行魅力（游说）检定来决定买家的报价。角色也可以选择拒售，而不是浪费工作周再次尝试。使用魔法物品基础价格表和魔法物品报价表来决定具体销售价格。 表：魔法物品基础价格 | | | |---------------------------------------------------------------|---------------------------------------------------------------------------| | 物品稀有度 | 基础价格 | | 普通 | 100gp | | 非普通 | 400gp | | 珍稀 | 4,000gp | | 极珍稀 | 40,000gp | | 传说 | 200,000gp | 表：魔法物品报价 | | | |-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------| | 检定值 | 结果 | | 1-10 | 50%的基础价格 | | 11-20 | 100%的基础价格 | | 21+ | 150%的基础价格 | 难题 出售魔法物品的最大难题是会引来小偷、骗子和其他任何想要东西但不想付钱的人。其他群众或许也会试图破坏交易以保护自己的业务，或者诋毁角色是一个不合法的卖家。 角色进行出售魔法物品的每个休整期都有10%的机会引发难题，出售魔法物品难题如表中示例。 表：出售魔法物品难题 | | | |---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------| | d8 | 难题 | | 1\* | 角色的敌人偷偷安排购入这件物品并用于对付角色。 | | 2\* | 物品被盗。 | | 3\* | 你的竞争对手散布这件物品是假货的谣言。 | | 4 | 有个术士宣称他对这个物品有与生俱来的所有权，并要求角色们交给他。 | | 5\* | 这件物品的旧主，或者他幸存的盟友，发誓要用力量夺回它。 | | 6\* | 买家在买之前被谋杀。 | | 7 | 买家凶恶的声誉彰显了这个物品一定会被拿去用于犯罪。 | | 8\* | 这件物品是神圣秩序失落的遗物。 | \*可能涉及对手
