太空仓鼠

太空仓鼠Space Hamsters

太空仓鼠的外形和它们的地表近亲类似，无论是外貌还是习性，而且它们的颜色也是繁复多样。

巨型太空仓鼠Giant Space Hamster

巨型太空仓鼠是一种如同棕熊一样巨大的超可爱啮齿类食草动物。它们往往会避开其他生物，尤其是具有威胁性的。但它们也可以被训练成坐骑或驮兽。侏儒曾经试图建造由巨型太空仓鼠转轮驱动的魔法船，但迄今为止仍然未告成功。

巨型太空仓鼠Giant Space Hamster

大型野兽，无阵营

AC：11

HP：22（4d10）

速度：30尺，掘穴10尺

力量14（+2） 敏捷12（+1） 体质10（+0）

智力2（-4） 感知12（+1） 魅力4（-3）

感官：被动察觉11

语言：-

挑战等级：1/4（50 XP）熟练加值：+2

动作

啃咬Bite。近战武器攻击：命中+4，触及5尺，单一目标。命中：7（2d4+2）点穿刺伤害。

太空仓鼠Space Hamster

第一批太空仓鼠是法师们用魔法将巨型太空仓鼠缩小到迷你的尺寸所制造出来的，因此它们的创造者称它们为迷你巨型太空仓鼠。这种魔法还让太空仓鼠变得更聪明，甚至能够心灵感应。

这些温和的啮齿类动物是产自荒宇的生物，但有不可计数的成员已经找到了前往物质位面其他世界的通路，在那里，它们直接被称之为仓鼠。它们还会小心翼翼地不让附近的大多数生物注意到它们能够进行心灵感应。

太空仓鼠Space Hamster

微型怪兽，普遍中立善良

AC：15

HP：10（4d4）

速度：20尺，掘穴5尺

力量1（-5） 敏捷20（+5） 体质10（+0）

智力6（-2） 感知12（+1） 魅力6（-2）

感官：察觉+3，隐匿+7

感官：被动察觉13

语言：心灵感应5尺

挑战等级：1/4（50 XP）熟练加值：+2

动作

啃咬Bite。近战武器攻击：命中+7，触及5尺，单一目标。命中：1点穿刺伤害。

打它眼睛Go for the Eyes*（充能6）。近战武器攻击：命中+7，触及5尺，单一生物。命中：8（1d6+5）点穿刺伤害，并且目标必须成功通过一次DC15的敏捷豁免，否则直到仓鼠的下个回合开始前陷入目盲。

附赠动作

撤退Escape。仓鼠使用疾走或撤离动作。

【*校注：游侠明斯克指挥他的仓鼠伙伴时常用的口号之一。】
