家族代理人

家族代理人 House Agent

属性值：力量、智力、魅力
专长：幸运
技能熟练：调查与游说
工具熟练：选择一种工匠工具
装备：选择A或B：(A)工匠工具（同上所选）、高档服装、20GP；或(B)50GP。

你与某个龙纹家族有着千丝万缕的联系，但你至今（尚）未显现出任何龙纹。你可能出身于家族血脉，也可能只是一名与其并无血缘关系的家族雇员。长久以来，你一直奔走于家族业务的第一线以谋求生计，如同家族高层延伸至外部世界的耳目与手足，忠实地执行着各项指令。你或许在你效力的家族内部有着不少旧友、导师或竞争对手；而在其他龙纹家族中，你同样有着不少盟友——或敌人。
