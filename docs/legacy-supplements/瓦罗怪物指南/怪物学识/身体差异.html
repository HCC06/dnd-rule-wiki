**身体差异 Physical Variations** ------------------------------------------------------------------------ 狗头人的鳞片颜色和形状各种各样。尽管人类可能无法分辨出两个长得很像的狗头人，它们自己却能轻易分辨出彼此。 同一部族的狗头人通常有着相似的色彩花纹。例如，铜齿 Copper Tooth 部族大多有着灰色间红条纹的外形。两个部族融合之后杂交的个体将形成一个新的外观，除非出现外来者或者某些出现其中某个原部族特征的返祖个体。 使用表格“鳞片颜色”随机决定狗头人部族的主要颜色外观。如果投骰结果指向花纹外观，则按表格“鳞片花纹”决定两个颜色如何组合。 **鳞片颜色 Scale Color** | | | |-----------------------------------------------------------------------------------|------------------------------------------------------------------| | **d100** | ** 鳞色** | | 01-05 | 黑 | | 06-10 | 蓝 | | 11-25 | 棕 | | 26-30 | 灰 | | 31-35 | 绿 | | 36-40 | 橙 | | 41-55 | 橙－棕 | | 56-60 | 红 | | 61-75 | 红－棕 | | 76-85 | 褐 | | 86-90 | 白 | | 91-00 | 花纹（骰两次并忽略复合色和91及以上的骰值） | **鳞片花纹 Scale Pattern** | | | |---------------------------------------------------------|--------------------------------------------------------| | **d20** | ** 鳞片花纹** | | 1-4 | 斑驳 | | 5-8 | 网格 | | 9-12 | 渐变 | | 13-16 | 斑点 | | 17-20 | 条纹 |
