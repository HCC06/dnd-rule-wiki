扮演一名兽人

**扮演一名兽人ROLEPLAYING AN ORC ** ------------------------------------------------------------------------ 大部分兽人都被教导过着破坏与屠戮的生活。但与那些天性邪恶的生物不同，比如鬣狗人，如果一个兽人在自己的文化之外被养大，能够成长出有限的同情，爱和怜悯。 无论一个兽人看上去多么驯服，它的杀戮欲望总是在其表面下流淌。兽人本能地热爱战斗，渴望证明自己的力量。想要在文明的范围内生活，一个兽人会面临一项艰巨的任务。 **** **兽人个人特质ORC PERSONALITY TRAITS** | | |
