**狗头人的名字 Kobold Names** ------------------------------------------------------------------------ 狗头人的命名都来自模仿龙类的发音，且通常都会跟其所有者的个人特色相关，诸如鳞片的颜色、特别的身体部位或者典型的习惯等。例如“红脚 Red Foot”、“白爪 White Claw”和“飞毛腿 Scurry”都是常用名字的通用语译名。狗头人成年时可能会更改自己的名字，也可能会在经历某些重要事件后在其名字里加上某些音节，这些事件可能是完成第一次狩猎，产下第一枚蛋，或者在第一场战斗中存活，等等。表格“狗头人名字”列出了适合用于任何战役中的狗头人名字。 **狗头人名字 Kobold Names** | | | |------------------------------------------------------|------------------------------------------------------------| | **d20** | ** 名字** | | 1 | 亚力士 Arix | | 2 | 依克丝 Eks | | 3 | 埃特 Ett | | 4 | 加莱克斯 Galax | | 5 | 加鲁 Garu | | 6 | 哈格纳 Hagnar | | 7 | 霍克斯 Hox | | 8 | 伊尔托斯 lrtos | | 9 | 卡沙克 Kashak | | 10 | 米波 Meepo | | 11 | 莫洛 Molo | | 12 | 欧索斯 Ohsoss | | 13 | 洛托姆 Rotom | | 14 | 萨格因 Sagin | | 15 | 希克 Sik | | 16 | 斯尼夫 Sniv | | 17 | 塔克拉克 Taklak | | 18 | 泰斯 Tes | | 19 | 乌拉克 Urak | | 20 | 瓦恩 Varn |
