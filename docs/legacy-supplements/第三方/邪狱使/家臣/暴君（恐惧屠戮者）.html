暴君（恐惧屠戮者）

##### 暴君Tyrant（恐惧屠戮者）
中型类人，任意阵营
------------------------------------------------------------------------ **护甲等级** 18（重甲） **生命值** 八倍于其等级的值（等于其等级数的d10生命骰） **速度** 30尺 ------------------------------------------------------------------------ | | | | | | | |----------|----------|----------|----------|----------|----------| | **力量** | **敏捷** | **体质** | **智力** | **感知** | **魅力** | | 16(+3) | 10(+0) | 10(+0) | 10(+0) | 10(+0) | 14(+2) | ------------------------------------------------------------------------ **豁免** 所有+PB **技能** 运动+3+PB，威吓+2+PB **感官** 被动察觉10+PB **语言** 通用语，炼狱语 **熟练加值（PB）** 等于其指引者的加值 ------------------------------------------------------------------------ **招牌攻击：长剑Longsword。** *近战武器攻击：* +3+PB，触及5尺，单一目标。*命中：* 1d8+PB挥砍伤害。 7级起，暴君在其回合内执行攻击动作时，可以发动两次此攻击而非一次。 ###### 特性Features **3级：军列移转Shift Rank（3次/日）。** 以一个动作，暴君发动一次招牌攻击且允许至多三名位于其30尺内能听见其的盟友移动至多10尺（无需动作），此移动不会引发借机攻击。 **5级：蹂躏Devastation（3次/日）。**以一个动作，暴君发动一次招牌攻击，并且至多三名位于其30尺内能听见其的盟友可以使用他们的反应发动一次武器攻击。 **7级：扼窒Choke（1次/日）。**以一个动作，暴君发动一次招牌攻击，若此攻击命中，暴君将以炼狱魔法扼住目标。目标陷入擒抱（逃脱DC10+暴君PB），且目标陷入束缚直至擒抱结束。
