幽魂蛛

精魂蜘蛛（浸泡在葡萄酒中的丝线）

栖息地： 森林，沼泽，地下；宝藏：无

悄无声息，难以察觉，幽魂蛛在最黑暗的角落活跃着，一面躲藏一面观察。它们小巧的、幽灵般的类蛛身躯轮廓模糊，像烟雾般近乎透明，而它们的节肢纤长而又参差不齐，结实得令人不适。它们编织的网是转瞬即逝却令人毛骨悚然，无论距离多远，幽魂蛛都能通过它们感知信息。

特质Traits

转瞬之躯Ephemeral Body。幽魂蛛可以移动穿过最窄1寸宽的空间而无需消耗额外的移动力。
先见之丝Seer's Strand。幽魂蛛可以通过以太蛛网所在的空间看和听，如同其位于该空间。
蛛行Spider Climb。幽魂蛛可以在难以攀爬的表面上攀爬，包括沿着天花板移动，且无需为此进行属性检定。
蛛网行者Web Walker。幽魂蛛在蛛网上移动时无视蛛网造成的移动限制，且与蛛网接触期间，幽魂蛛可以知道其他与该蛛网接触生物的准确位置。

动作Actions

多重攻击Multiattack。幽魂蛛发动两次啃咬攻击。
啃咬Bite。 近战攻击检定：+5，触及5尺。 命中：5（1d4 + 3）穿刺伤害，且若目标为生物，其陷入中毒状态，直至幽魂蛛的下个回合开始。
以太蛛网Ether Webs （2/日）。幽魂蛛以隐形的、暂时性的蛛网充满5尺立方区域，持续8小时。如果蛛网没有被固定在两个固体中间（例如墙和树）或跨在一片地板、墙壁、天花板中间，则蛛网会自动坍塌并消失。当一名生物触碰蛛网时，只要它们位于同一存在位面，幽魂蛛就知道该生物的准确位置。
