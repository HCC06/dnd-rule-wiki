奥姆城诡神

艾金I'JIN

（读作EYE-jin）

独角兔艾金，善变而难以捉摸。

缺点

当你被艾金附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我从不按计划行事。”

能力

当你被艾金附体时，你的敏捷值变成23，除非它已经更高。

库巴赞KUBAZAN

（读作 KOO-bah-zahn）

沼蛙怪库巴赞，狂野而充满活力。

缺点

当你被库巴赞附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我无所畏惧，风险再大我也敢试一试。”

能力

当你被库巴赞附体时，你的力量值变成23，除非它已经更高。

莫阿MOA

（读作MOH-ah）

幻色蛇莫阿，诚实而善良。

缺点

当你被莫阿附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“无论任何时候，我都只说真话。”

能力

当被莫阿附体时，你可以使用一个动作来隐形。你正在持用和携带的所有物件都将与你一同隐形。如果你攻击、施法、强迫他人进行豁免或造成伤害，该隐形效果将结束。

娜浪NANGNANG

（读作NANG-nang）

格龙蛙人娜浪，自私而残忍。

缺点

当你被娜浪附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我不会与他人分享。”

能力

当你被娜浪附体时，你可以在垂直表面和天花板上自由移动，这无需占用你的双手。你还将获得与你的步行速度相同的攀爬速度。

奥博娜卡OBO'LAKA

（读作oh-boh-LAH-kah）

佐博怪奥博娜卡，胆怯而偏执。

缺点

当你被奥博娜卡附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我厌恶风险，只会循规蹈矩。”

能力

当你被奥博娜卡附体时，你可以额外同调一个魔法物品。当奥博娜卡离开你时，你失去与所有魔法物品的同调。

帕帕扎图PAPAZOTL

（读作pah-pah-ZAH-tul）

魔王鹤帕帕扎图，精明而狡诈。

缺点

当你被帕帕扎图附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我不听任何人的意见，只希望别人都照我的命令去做。”

能力

当你被帕帕扎图附体时，你不会被突袭，同时你所有基于感知的属性检定具优势，并且你不会受到任何坠落伤害。

莎根比SHAGAMBI

（读作shah-GOM-bee）

六岐蛇豹莎根比，明智而善良。

缺点

当你被莎根比附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我对作恶者从不手下留情。”

能力

当你被莎根比附体时，如果你在自己回合内采取攻击动作，则你可以额外进行一次攻击。

安珂UNKH

（读作UNK）

链枷蜗牛安珂，自私自利而优柔寡断。

缺点

当你被安珂附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我无法做出任何决定。”

能力

当你被安珂附体时，你的体质值变成23，除非它已经更高。

望戈 WONGO

（读作WONG-go）

苏怪望戈，暴力而疯狂。

缺点

当你被望戈附体时，你获得以下缺点，它覆盖任何与之冲突的缺点:“我做事从来不顾他人死活。”

能力

当你被望戈附体时，你可以用动作对周围60英尺内你可以看见的一个生物发动一次心灵打击。目标必须进行一次DC 16的感知豁免，豁免失败则陷入震慑，直至其下回合结束时。
