第六章：自定义选项

第六章：自定义选项Customization Options
属性值、种族、职业以及背景等各选项结合起来定义了你的角色在游戏中能做到的事情，而你在创作角色时对各细节的选定则使你的角色更加独一无二。通常在选定职业和种族后，角色的能力大小已经大体定型。而本章的内容则允许玩家在角色设定完成后，遵照DM指示为角色作进一步调整。

本章定义了两组自定义角色所使用的可选规则：兼职和专长。兼职允许你组合不同的职业，而专长则提供了一些选项，用以替代升级时获得的属性值提升机会。而是否在战役中启用这些可选规则则由你的DM作最终的判定。
