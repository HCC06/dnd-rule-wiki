老兵

老兵 Veterans

老兵是专业的战士，他们重新拿起武器只是为了捍卫自己的某种信念与价值，或是单纯为了一份相称的报酬。这些人包括从长期兵役中退下的士兵，以及那些只为自己卖命的武者。

老兵Veteran
中型类人（任意种族），任意阵营
护甲等级：17（板条甲）
生命值：58（9d8+18）
速度：30尺
力量16（+3） 敏捷13（+1） 体质14（+2）
智力10（+0） 感知11（+0） 魅力10（+0）
技能：运动+5，察觉+2
感官：被动察觉12
语言：任意一门语言（通常是通用语）
挑战等级：3（700 XP）

动作
多重攻击Multiattack。老兵发动两次长剑攻击。如果其短剑已出鞘，则可再发动一次短剑攻击。
长剑Longsword。近战武器攻击：命中+5，触及5尺单一目标。命中：7（1d8+3）点挥砍伤害，或使用双手攻击时8（1d10+3）点挥砍伤害。
短剑Shortsword。近战武器攻击：命中+5，触及5尺，单一目标。命中：6（1d6+3）点穿刺伤害。
重弩Heavy Crossbow。远程武器攻击：命中+3，射程100/400尺，单一目标，命中：6（1d10+1）点穿刺伤害。
