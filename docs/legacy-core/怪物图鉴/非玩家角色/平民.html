平民

平民 Commoners

各种平民包括乡民、农奴、奴隶、仆从、朝圣者、商人、工匠，还有一些普通隐士。

平民Commoner
中型类人（任意种族），任意阵营
护甲等级：10
生命值：4（1d8）
速度：30尺
力量10（+0） 敏捷10（+0） 体质10（+0）
智力10（+0） 感知10（+0） 魅力10（+0）
感官：被动察觉10
语言：任意一门语言（通常是通用语）
挑战等级：0（10 XP）

动作
短棍Club。近战武器攻击：命中+2，触及5尺，单一目标。命中：2（1d4）点钝击伤害。
