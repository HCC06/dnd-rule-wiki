体型 Size

怪物的体型可以是微型Tiny、小型Small、中型Medium、大型Large、巨型Huge或是超巨型Gargantuan。体型分类列表展示了每只特定体型怪物在战斗中所占据空间的具体大小。《玩家手册》中可以获得关于生物体型和空间的更多信息。
