紫腐菌

紫腐菌Violet Fungus

这种紫色的蘑菇用长在其底部的根状触须在洞窟地面上爬行。紫腐菌从伞盖中央向四周辐射状长出鞭状的茎用以抽击敌人。其轻微的触碰就可以使肉体迅速腐烂，也正因此紫腐菌杀死的生物尸体可以迅速完成分解。新的紫腐菌会在腐烂的尸体中长出，并在2d6天内成长完全。

紫腐菌Violet Fungus

中型植物，无阵营****

护甲等级：5

生命值：18（4d8）

速度：5尺

力量3（-4） 敏捷1（-5） 体质10（+0）
智力1（-5） 感知3（-4） 魅力1（-5）

****

状态免疫：目盲，耳聋，恐慌

语言：———

挑战等级：1/4（50 XP）

****

拟形*False Appearance。***静止的紫腐菌与普通的真菌一模一样。

动作

多重攻击*Multiattack。***紫腐菌发动1d4次腐烂触击攻击。

腐烂触击*Rotting Touch。*近战武器攻击：命中+2，触及10尺，单一生物。命中：4（1d8）点暗蚀伤害。
