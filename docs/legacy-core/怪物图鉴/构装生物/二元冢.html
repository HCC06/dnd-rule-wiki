二元冢

二元冢Duodrone

块状的二元冢可以监控单元冢的任务，并且可以同时执行两项任务。

二元冢Duodrone

中型构装，守序中立

护甲等级：15（天生护甲）
生命值：11（2d8+2）
速度：30尺。

力量11（+0） 敏捷13（+1） 体质12（+1）
智力6（-2） 感知10（+0） 魅力7（-2）

感官：真实视觉120尺，被动察觉10
语言：魔冢语
挑战等级：1/4（50 XP）

公理之念*Axiomatic Mind。***二元冢不会被迫去做违反其天性或其原有指令的行为。

解体*Disintegration。***二元冢死亡时，它的身体会瓦解为尘埃，留下它的武器和其他携带的东西。

动作

多重攻击*Multiattack。***二元冢发动两次拳击攻击或两次标枪攻击。

拳击*Fist。*近战武器攻击：命中+2，触及5尺，单一目标。命中：2（1d4）点钝击伤害。

标枪*Jacelin。*近战或远程武器攻击：命中+3，触及5尺或射程30/120尺，单一目标。命中：4（1d6+1）点穿刺伤害。
