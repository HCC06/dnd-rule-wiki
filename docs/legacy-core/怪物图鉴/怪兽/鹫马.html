骏鹰

骏鹰Hippogriff

这种野兽不可思议的起源已经无从考证，一匹骏鹰长着鹰的翅膀和前肢，马的后腿以及一颗结合这两种动物特征的脑袋。
生性隐匿而杂食的骏鹰，带着其唯一的终生伴侣且很少冒险离巢超过几里。它们会为保卫配偶或幼崽而奋战至死。骏鹰不下蛋而是胎生。
龙，狮鹫和飞龙喜欢骏鹰肉的味道而时常捕食这种生物。

飞行坐骑*Flying Mounts。***笼中饲养的骏鹰可以被训练为忠诚的伙伴和坐骑。在所有可以驯养为飞行坐骑的生物中，训练得当的骏鹰最易培养且最为忠诚。

骏鹰Hippogriff

大型怪兽，无阵营****

护甲等级：11

生命值：19（3d10+3）

速度：40尺，飞行60尺

力量17（+3） 敏捷13（+1） 体质13（+1）
智力2（-4） 感知12（+1） 魅力8（-1）

技能：察觉+5

感官：被动察觉15

语言：———

挑战等级：1（200 XP）

敏锐视觉*Keen Sight。***骏鹰依赖视力而进行的感知（察觉）检定具有优势。

动作

多重攻击*Multiattack。***骏鹰发动两次攻击：一次喙啄，和一次爪击。

喙啄*Beak。*近战武器攻击：命中+5，触及5尺，单一目标。命中：8（1d10+3）点穿刺伤害。

爪击*Claws。*近战武器攻击：命中+5，触及5尺，单一目标。命中：10（2d6+3）点挥砍伤害。
