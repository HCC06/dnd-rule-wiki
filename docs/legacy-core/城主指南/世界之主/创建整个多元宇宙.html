塑造多元宇宙

冒险者随着等级提升其冒险也扩展至另一现实维度——即多元宇宙的各存在位面。角色们可能应召到恐怖的无底深渊Abyss中解救一名朋友，或是于海神之河River Oceanus闪光的水面上航行。他们将在伊斯加德Ysgard与友善的巨人们举行宴会，再去混沌海Limbo直面一位身形瘦削的吉斯泽莱智者。 诸存在位面定义了一些极其怪异而危险的环境。其中那些匪夷所思的处所甚至完全超乎自然世界的料想范围。位面冒险可提供与普通状况迥然不同的惊异危机。而冒险者们将在固体火焰构成的街道上行走，或在每个黎明与无限复活的亡者们在战场上以战斗自证其身。
