多种稀有度（法术卷轴）

法术卷轴 Spell Scroll
卷轴，多种稀有度
每张法术卷轴都以神秘暗语记录了一道法术。如果该法术属于你的职业法术列表，则你可以阅读该卷轴并施展其法术且无需提供任何材料成分。否则你将无法理解该卷轴。通过阅读卷轴施展的法术同样需要遵循相应法术正常的施法时间。法术施展过后，卷轴上的文字将随之消散，卷轴本身也将化作尘土。施法被打断时，卷轴并不会因此消失。
假如卷轴记载的法术在你职业的法术列表内，但其环阶超过了你目前能施展的水平，则你必须以你的施法属性通过一次属性检定，DC等同于10＋卷轴法术的环阶。如果检定失败，则卷轴上记载的法术将消失而不生效。
卷轴上的法术一经施展即消失。卷轴本身则化为尘埃。
卷轴的稀有度、其法术的豁免DC和攻击加值由它所记载的法术环阶决定，如下表所示：

如同抄录法术书上的法术一般，法师也可以抄录记载在法术卷轴上的法师法术。抄录卷轴上的法术时，抄录者必须成功通过一次DC为10＋法术环阶的智力（奥秘）检定以成功抄录。无论检定成功与否，抄录后，法术卷轴都将被摧毁。
