重伤

**重伤** **Massive Damage**

该选用规则让生物更容易被重大伤害击倒。
一个生物从一次单个来源中承受的伤害大于或超过其生命值上限一半时，它必须进行一次DC 15的体质豁免，失败则依照表格“休克”掷骰并承受相应效应。例如，如果一个生物的生命值为30，那么当它受同一来源的一次伤害达15时即满足该条件。
